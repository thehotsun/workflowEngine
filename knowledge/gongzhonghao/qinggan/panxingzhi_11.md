# 一场家庭风波使我离开了喜爱的行业，现在很后悔怎么办

# 控制不住自己打孩子，我该怎么办？

哪个更重要？在我父母尚未离婚的时候，我的家就像一个炮火纷飞的战场。我的父母可以为了任何一个人，任何一句话，任何一件鸡毛蒜皮的琐事吵得不可开交。吵到激烈之处还会对家里的锅碗瓢盆连摔带砸，完全顾不上躲在角落里瑟瑟发抖的我。 记得有一次放学回家，恰逢父母正吵得天翻地覆，家里传来一阵阵让人毛骨悚然的尖叫声和东西破碎的巨响，我茫然无措地站在门口，内心挣扎着要不要进去，要不要回家，要不要面对这俩可怕的恶魔。 那是一个冬天的傍晚，外面飘着鹅毛大雪，我在寒风中瑟瑟发抖，饥肠辘辘。但是无论我如何敲门叫喊都无人理睬，我的父母沉醉于互相争吵和摔东西的快感中，完全遗忘了快要在寒风中冻僵的我。 后来好心的邻居收留了我，给了我一杯热水和一顿暖心的晚餐。邻居是一对和善又温柔的夫妇，女儿和我在一个学校念书，女主人做的饭菜非常好吃。 从此每天我都会在邻居家吃饭、写作业，待到很晚才回家睡觉。比起我那个炮火纷飞真正的家，邻居的家像一个宁静和谐的避风港，给了战乱里的我一丝温暖和希望。 说实话，跟那个真正但是炮火纷飞的家比起来，我更想要一个能够让我感到安全和温暖的环境，哪怕它并不完整。02"都是为了你"VS我要尊严和权利 后来我父母终于离婚了，虽然我的世界安静下来了，但是紧接着我的殖民地生涯又开始了。我的妈妈习惯于“为我好”，她会事无巨细地给我安排好一切，而我需要做的就是严格按照她规划好的道路去走就行了。就连文理分科，高考填志愿都是我妈去给我填的。 印象深刻的是初中时候，我因为考试失利，害怕妈妈伤心，便自作主张伪造了一份假的成绩单。这件事被我妈发现之后，我迎来了自己人生中最惨痛的一顿毒打，长达两个小时。 无论我怎样哭泣和哀求都平息不了妈妈的怒火。她的疯狂和怨恨让我深深的恐惧，直到多年之后我才知道，那是一种深深的焦虑和不安，她无法面对自己失败的婚姻和人生，只能通过控制我、占有我来实现。 曾有媒体报道过某孩子被父母活活打死的事例，很多人无法相信哪里会有亲生父母如此狠心的。其实我是由衷地相信他的父母是真心为了他好的。但是这种畸形扭曲的爱，往往会酿造骇人听闻的悲剧。 对于我的妈妈来说，一切只是因为她婚姻失败了，她感受到了巨大的挫败感，于是我不幸沦为了她怒刷存在感的唯一对象，她剥夺了我想要过自己人生的权利，而把自己想要的人生强加给了我。 其实我真心不想她做什么都是为了我好，真心不想她牺牲奉献就是为了我，真心不想活的这么压抑这么沉重。 可是为什么她就是不明白，我不是她的所有物也不是她的殖民地，我有我自己的人生和幸福，而她也要找到除了孩子之外的人生价值和意义。 我理解妈妈内心深处的孤独寂寞，但是我不该成为她圣母式付出的对象和无底线精神的寄托，不该被套上巨大的枷锁，我唯一希望的是，她能够找到自己人生的意义并获得幸福。 我们繁衍后代不是为了把孩子变成自己的一部分或是形成一种永久性的控制关系，而是一条父母与孩子共同成长、共同进步的漫漫长路。 03爱我，请先和我好好说话从小到大，我的父母都口口声声说很爱我，然而却没有一个人肯和我好好说话。他们的态度永远居高临下，口气永远冰冷生硬，话语永远挖苦讽刺。 然后再用心良苦地说，我之所以批评你、指责你、教训你甚至揍你，都是因为我爱你。爱难道就是批评、指责和教训？ 我童年最深刻的记忆就是端饭的时候不小心把碗摔碎了，我爸会立即一个耳光甩过来：这么点小事都做不好你能干嘛？洗澡的时候不小心让毛巾掉在地上了，我妈又会指着我的头说：你不知道这地上很脏吗？ 就这样，我陷入了一种负性的循环之中，父母指责—我紧张—出错—越发指责—越发紧张—越容易出错。直至后来我就像是一根永远紧绷的弦，时时刻刻处于紧张和戒备的状态。并且因为害怕被责备，所以我形成了一副委曲求全的性格。我表面风光，内心却自卑懦弱，胆小怕事。我做事完美，心理却消极、厌世，缺乏阳光。 我和第一个男朋友交往的原因很简单。有一天，我不小心摔碎了他心爱的杯子，看着我惊慌失措的样子，他只是轻描淡写地说了一句：没事啦，杯子碎了再买一个就是了。 然后我突然感到了一种前所未有的安心和平静。是啊！东西摔坏了，买一个新的就好了。考试没考好，下一次努力就可以了。人生除了生死，真的没有特别重要的事情，为什么就不能多一些宽容，少一些愤怒呢？ 就如同我父母的婚姻，貌似毁于婚外情，其实本质源于两个人都不懂得好好向对方说话，更不懂得相互欣赏和支持的意义。 04爱我，请和我一起成长从小到大，我都是个乖巧懂事的孩子，我很理解父母望子成龙的心情，也一直在努力让自己变得更好。但是令我遗憾的是，我的父母一边拿着鞭子赶着我一路狂奔，一边自己却心安理得做着没有任何理想和追求的人。 我的妈妈严格规定我每次考试必须名列前五，而她自己却在一份毫无起色的工作上一干就是三十多年；她要求我空闲时间都要用来看书和学习，而她却把空闲时间全部交给了电视。 就像曾经看过的一副漫画：一对夫妻在麻将桌上搓麻将，却指着书本对桌边的孩子说，你赶紧看书去！这是多么大的讽刺！ 就像当下的婚姻，我们一直在强调夫妻之间的共同成长，却忽略了其实父母与孩子之间更应该共同成长。 在大人眼里，小孩子家家的懂什么，吃过的米还没有你们吃过的盐多。但是孩子和大人所不同的是，他们一直在进步，一直在提升，一直在成长。 父母却始终意识不到孩子和自己渐行渐远的根源所在，如果你们只是安于让自己的孩子成为精神寄托而放弃了自己的成长和进步的话，那么你的孩子就真的会离你越来越远，直到成为一颗你永远无法触及的星星。05写在最后 去年在杭州灵隐寺祈福的时候，看到身边有一个四五岁的小女孩也在虔诚地祈祷。 我很好奇地问：你在许愿吗？她很认真地说，我想让神仙把妈妈变得漂亮一些，否则我会被其他小伙伴嘲笑。 可能很多人都要吐槽：父母把你养这么大，你却嫌弃自己的父母不好看，真是个熊孩子！ 事实却并非如此。孩子是父母的一面镜子，你嫌弃他，他也会嫌弃你；你抱怨他，他也会抱怨你；你夸别人家的孩子好，他夸别人家的妈妈好；你不好好和他说话，他也不会和你好好说话；你不懂得尊重他，他也绝不会发自内心地尊重你。 或许也会有人吐槽我，自古以来就是君君臣臣父父子子，家长就代表着权威。但是权威也要懂得尊重别人，权威也要讲道理，权威也要懂得好好说话，权威也不是绝对的正确。 相信我，比起离婚、单亲、父爱、母爱，孩子更缺的是尊重、理解、赞赏和支持。如果你没在深夜读过潘幸知如果你不曾为爱痛哭过谈何人世走一遭？关注我 # 没有享受过母爱的人，如何在新家庭中给到孩子母爱？


没有享受过母爱的人，如何在新家庭中给到孩子母爱？我有一个来访者，她的母亲因为生病只能长期卧在病榻。母亲身体里有很多的气，就变成了攻击女儿的力量。母亲指挥女儿在很小的年纪，就做很重的家务，一面喜欢干净，一面自己无法动弹，就指挥女儿干活。这个小小的女孩，一边抗争一边干活。在家教里，男孩子是跟着爸爸走天下的，女孩子是要跟着妈妈做家务的。所以，小小的她承担了原本不该属于她的一切，成绩很好，但回到家迎接她的，不是掌声，不是“你真棒”，而是，赶紧做饭洗衣服。常年爸爸对她是忽略的，因为母亲生病，爸爸要在外打拼赚钱，后来哥哥也结婚了。她更孤苦无依，她就想脱离家庭，她恨妈妈，又心疼妈妈。如今那个小小的她长大了，她也有了自己的女儿，只是面对女儿的时候，似乎又重复了母亲的模式，在跟女儿的沟通中，她感觉很无力。对女儿的无力，对母亲的无能为力，还有愧疚，一直久久的折磨着她，无法自拔。她说，幸知老师，我11岁开始就没有母爱了。我说，母爱她一直在，只是妈妈连爱自己的能力都没有，所以就一点点蚕食了女儿的爱，来给养自己。这个小女孩长成了“母亲”，但是如果她很有力，很开朗，那是对她母亲的背叛。因为病榻的母亲，受着气的母亲，伴随着她阴暗的童年成长，是她挥之不去的模式。如果成年后的她那么快乐，就是对家庭的背叛，因为她快乐，就会觉察到母亲的悲伤，她越快乐，就越背叛愁苦的母亲。而这愁苦的母亲，就是她根深蒂固的根儿。所以，她要用愧疚，来证明自己没有背叛，证明自己是道德的。这就是无力感的来源。那怎么办呢？先完成背叛——和原生家庭的脱离，再进行孝顺——和原生家庭的和解。在脱离原生家庭之前，先要“看见”。看见自己的“大爱”，认可自己的“大爱”。11岁的年纪，本应该好好被爱，可以做一个在母亲怀里恃宠而骄的小公主，但是这么小的年龄却承受和背负了不属于自己的这一部分责任，那么那时候对母亲的“怨恨”和“对抗”都是应该的，顺理成章的啊，先看到这样的自己，为家庭里付出了那么多那么多的自己，那个没能好好享受童年的自己，在这个本该索要爱的年纪，做到了超出一个正常孩子的能力边界，甚至把自己身体的一部分无私的奉献给了家庭。这是多么大的爱啊。这样优秀的孩子，让我们回到童年去看见她，抚摸她，心疼她，发自内心的赞美她，对她说一句“你太不容易了”。母亲有母亲的命运。我有我的命运。虽然我们母女连心，但是我只有先努力有权力让自己舒服，才能更好的照顾母亲和女儿。我可以无视我的母亲现在的痛苦，这不意味着不孝，也许我可以一周有一次与母亲的痛苦链接，但是其他时间，我可以允许自己是非常非常快乐的，快乐得像黄鹂一样。我可以“没有无力感”，这不是不道德，这是为了更好的帮助母亲，因为一只快乐的黄鹂，一定会把快乐的情绪传递给母亲，哪怕此刻没有。但如果我们是只悲伤的乌鸦，那么给到母亲的，可能无论是此刻，还是未来，都是悲伤。看见自己的内在小孩，认可自己的内在小孩，我们长大了，我们有能力做自己内在小孩的爸爸和妈妈。保护好自己，是第一步的爱。如果我们缺爱，又如何能真正给到女儿爱呢？就只会在不知不觉中重复母亲的模式，以爱的名义，掠夺控制女儿来滋养自己，那么双方都会是悲伤的。做完这些内在的清理，我们才能真正的接纳母亲，接纳她不是一个理想母亲，接纳她不是一个完美母亲，她甚至不懂得如何好好的爱女儿，但是她一直在，是我们的根儿。我活得好，就是她最大的成果。

# “没什么不满”的婚姻，怎就出现了重大背叛？


有一位来访者，刚生完孩子没多久，坚持要跟丈夫离婚。她坐月子期间跟老公发生了冲突。孩子又得了肺炎，一直抢救。这一切让她觉得，生孩子的苦丈夫无法理解，这个男人不但保护不了自己，还让给自己带来诸多伤害。加上产后的一些生理因素，她的情绪非常激动，坚决表示过不下去了。我帮她还原了冲突前后，另一个视角的生活场景：孩子出生，丈夫是第一次当爸爸；妻子产后虚弱，不能带孩子；家里来了老人帮忙，但老人也并不擅长照顾新生儿；丈夫需要协调家庭关系，帮忙照顾孩子；结果，孩子还得了肺炎……他的心理会是什么感受？婚姻是一个历程，孩子出生，是一个重要的站点，也是婚姻里的一个陷阱。猝不及防，俩人都掉进去了，都很无力、无助。我说：“你们两个人都受苦了。”她说：“老师，你这样说我舒服多了，原来这个陷阱里不是我一个人，他也痛苦，那我就舒服多了。”听起来，这位女士好像不通情理——怎么能因为丈夫也痛苦，她自己才能觉得舒服呢？但事实上，真正让她舒服的是：原来，我们两个人还是在一起的！掉入外遇陷阱，如何去正确解读关系外遇发生，同样也是婚姻经历的一次劫难，是两个人同时掉进一个陷阱。怎么办呢？只有在情绪调降的情况下，我们才能冷静思考：我要解决什么问题？我想要的是什么？想清楚之后，接下来你所做的一切，都是为了实现你想要的这个目标。有人说，我想要挽回。可是，情绪指引你去叫来所有的亲朋好友，以为人多就能说服对方，却反而把对方推得更远。有人说，我不知道我能要什么，反正就是不想让他好过；有人说，我想要把他从小三那夺回来……其实，这些并不是你真正想要的，只是对方的背叛行为激起了你的愤怒，主宰了你的行为，让你没有办法去考虑“我内心真正想要的是什么”。我们试着用新的翻译系统，来解读我们的关系。旧的翻译系统是这样解释的：背叛太可怕了，就是因为小三，我们的婚姻才会没有办法走下去的。我想要的是，我们的关系回到从前，我不要现在的痛苦。事实是：出轨不会是你们关系最本质的问题。问题在于，关系中的两个人并没有觉察到各自的需求变化，也没有学会把自己的需求变化分享给关系里的另一个人。事实是，两个人之间的情感连结早已经悄悄发生了变化。你需要的，也并不是回到从前，因为今天的结果就是从前的经历造成的。你真正需要的，是一段崭新的、安全的情感连结。很多遭遇背叛的人，并不知道从前你们关系的真实样子，更别提是不是你自己想要的关系。所以，遭遇背叛之后，真正需要的，是重新梳理关系。“没有不满”的婚姻，暗藏怎样的风险如果要对关系的满意度打分，0是不满意，10是非常满意。你会打几分呢？如果对关系的不满意度打分，0是没有不满，10是有非常多的不满。你会打多少分呢？有人可能会觉得，不满意度得分越低，是不是就是满意度越高呢？其实，这是两个完全不同的维度。有的婚姻没有什么不满意的，但一样不幸福。两个从没有发生过冲突，说不出来有什么不好，但满意度很低，同样说不出来有什么好的。影响婚姻关系的不满意度，是关系中冲突的频率、冲突的激烈程度和冲突的延续时间。换句话说，冲突越频繁发生，冲突越激烈，冲突时间越久，不满意度就越高。比如外遇的发生，我们对婚姻的不满意度就很高。如果一对夫妻十几年没说话，没接触，就很可能没有不满意，但同时也不会有满意。真正影响关系中满意度的，是关系中的亲密度，就是关系的情感连结程度。我们说经营婚姻，其实是要需要从两个方面来努力的：降低婚姻的不满意度，增加婚姻的满意度。在关系里将冲突的频率降低，不满意度就降低，这样才会产生新的希望或增加希望，才有可能增加亲密感的连结，希望感和改变的动力才会增加。关于婚姻满意，也有一部分人活在自以为是里，自我感觉满意，但另一半其实并不这么觉得。一般来说，女性是对关系比较敏感的，但依然会有很多女性容易陷入“自我感觉良好”的误区。原因在于她本身并不是那么在意关系，更在乎的是，对方提供的这种看似圆满的状态。有一位女性来访者，咨询的时候说，结婚十一年，孩子九岁了。突然有一天，先生提出要离婚，自己完全无法接受。“我们的婚姻一直挺幸福的，我是老师，他经营企业，对孩子也很负责，怎么会突然要离婚呢？他一定是被小三勾引了，只是他不承认……”当我让她对过去婚姻的满意度评分，她说7、8分。“没有什么不满意的，也有吵闹冲突，但谁家不吵呢？吵完过去就好了啊！”当我跟先生沟通的时候，他说：“我不是现在才想离婚的，早些年就想了，我努力了十年都没有改变。”先生工作非常努力，对孩子对家庭非常负责。他给婚姻的满意度打分是3、4分，不满意度是7、8分。这和太太的差距是非常大的。先生“努力了十年”，到底指的是什么？这位太太知道吗？在咨询过程中，我了解到，这对夫妻小时候都生活在父母争吵的家庭，但家庭环境对他们的影响却不同——先生非常讨厌父母争吵，曾经因为此离家出走，所以每次冲突他的感受都是无力和想逃，他总想着我多做一些，会不会好点；而太太会觉得父母虽然天天吵架，但床头吵架床尾和，第二天就好了。所以家里发生冲突，她也是这样，看先生也是觉得做的很好，所以她就不在意吵架的事了。同一件事，两个人各自的翻译系统是不一样的，看法感受就是不同的。在这种看似风平浪静的婚姻里，其实是潜藏了很多问题。转换翻译系统，提升对婚姻变化的敏感度在前面的案例中，如果我们可以换一个眼光去看婚姻，就会发现，这两个人其实都很不容易。每个人看待关系，就像有不同的翻译系统，或者说是戴着有色眼镜在看。换一个翻译系统，并不是为了要理解对方，至少我们要做到了解自己。先了解自己的需求，再来看关系里的另一半走到哪了。从恋爱到结婚，一切都是在变化的。我们是否可以看到这种变化？出轨方有一个共同的特点，就是都比较以自己的感受和需求为中心。婚姻生活趋于平淡之后，我们除了每天重复生活中的琐碎之外，也要留出一部分去感受自己，感受对方，让生活慢一点。这样，即便婚姻中出现了问题，也可以及时处理，而不至于突然崩溃。重大的背叛或冲突，看似都是突然降临，但实际上一切都酝酿已久。如果我们回顾感情的发展过程，就会更清楚这个翻译系统是怎样发生变化的。关系的初期，好像很少遇到问题。原因在于，一方面彼此迫切想取悦对方，以便可以留下美好的印象；另一方面，两个人也都带着爱意去翻译和解读对方的行为。当一些不可避免的问题显露的时候，我们在不知不觉中开始有了怨气，开始用敌意和怀疑去翻译和解读对方的行为，甚至开始互相指责或者恶语相向。慢慢地，我们开始不再彼此尊重，开始诋毁对方，而不是想办法弄清楚到底发生了什么，然后诚心诚意地坐下来沟通交流。当生活让我们越来越失望的时候，我们开始把所有的不开心都归咎于另一半，把他当成敌人来对待。外遇发生的时候，我们的翻译系统是分裂的。一方面，要面临突然的失去，我们想到他过去的好，想到孩子需要一个完整的家，想到他还不错……这时候，解读这件事就变成了忍辱负重，义无反顾地要挽回，而根本不去看：这还是我需要的那个人吗？我们之间的问题要怎么解决，我想好了吗？另一方面，我们开始带着悲观主义色彩去看各自的未来——如果离婚，我会更惨，未来可怎么办？我们还会带着毁灭性的愤怒去看这件事，我不好过，也不让他好过，等等。今天，新的翻译系统是：一切发生都不是最坏的，至少可以是一个信号和提醒。如果我们带着智慧和关爱去解决这些矛盾的话，这些事件发生后，智慧可以让我们更了解彼此，让我更了解自己。每个人的一生，都是不断成熟的过程。我们在这个过程中要完成的最重要的事，就是更全面地了解自己，不断增强自己的能力去拥抱自己的各种感受，这样我们才能得以成长，获得自主性以及变得完整。要实现这一切，我们必须增加自我觉察，即有能力去关注自己内心深处更加微妙的感受。我们把这样的能力叫做“个人觉醒”——对自身存在的各个方面都变得敏感，同时对自己周围的一切都充满同理心，包括对他人的需求和环境的变化。这需要我们去感受内心体验的能力。仅仅靠“头脑知识”，我们不可能变得更睿智成熟。虽然这样的知识也很有价值，但要想发展一段令人满意的感情，我们需要具备一种微妙的智慧——能够意识到彼此的内心感受。当我们愿意用新的眼光去看待自己、看待关系的时候，新的可能性就出现了。即便我决定放弃这段关系，也可以更坦然地面对自己。因为，经过这段遗憾，我更全面地了解自己，也愿意贴近自己、拥抱自己。一对一免费情感分析点这里↓

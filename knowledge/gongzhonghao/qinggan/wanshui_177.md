
# 有一种爱情，是在流水线上批量生产


从网友那里学到了一个新鲜词：PUA，全称Pick-up Artist，译成汉语就是“把妹达人”的意思。“把妹达人，起初指的是一群受过系统化学习、实践、和不断自我完善泡学（把妹）技巧的男人。后来泛指在很会吸引异性，让异性着迷的男女。在字面上的解释，PUA指的是‘搭讪艺术家’，但因为泡学文化的变迁和进步，PUA的定义已经从简单的搭讪扩展到整个吸引流程，主要包括：搭讪（认识）、吸引、建立舒适和联系、到发生亲密关系。”——以上是百度百科的解释。据说PUA起源于美国，得益于互联网的发展，自2008年起，一系列围绕着PUA文化的论坛也得以在中国获得了推广。“目前整个中国PUA圈子约有50万活跃成员，分布在各大城市”。这真是不看不知道，一看吓一跳。泡妞居然还有这么大名堂和花样，泡出了文化，建立了“泡学”，被冠上了“搭讪艺术家”的职称。想想早年统称为“耍流氓”的行为，现在有了组织、归属和文化定义，真是应该感谢这伟大的时代。不过就像很多外来文化最终会被同化一样，PUA文化的本意是让男性更具有个性魅力，学会怎样了解异性，并与女性友好相处，但它在中国前进的轨迹却还是“耍流氓”的老路。小静的初恋就是被这样一个国产“把妹达人”葬送的。她才20岁，是一个文静清秀的小姑娘，大一快结束的时候遇到了他。“他属于看起来很阳光的类型，高高瘦瘦的，运动方面很好，学习也不错，乐器书法也会一点”，这样的男孩子大概是大学中最受欢迎的类型了，她很快就在他的追求下动心了，投入了初恋的全部热情。“刚刚恋爱的时候，我什么都不懂”，以为两个人只要互相喜欢就是爱情了，她很在乎他，也很开心的付出着，非常珍惜彼此之间的感情。“我很有耐心，克服了自己的小情绪，一点也舍不得看到他不开心”，两个人在一起度过了一段蛮愉快的时光。或许是感受到了她的真情，知道她是一个好女孩，或许是觉得再没有必要隐瞒，不想玩游戏了，有一天，他选择了对她坦白，他是很有经验的PUA，“他说，他有过三个当真的女朋友，至于玩玩一类的，有8个几个的记不清了。而且，他和四个女孩那样在一起过，其中有三个女孩都是自己的第一次。”至于小静应该归属于哪一类，大概介于这两者中间吧。不算最当真的女朋友，但也不忍心玩玩就算。只能分了，她也不恨他，“怎么舍得去恨、去怪一个给过自己那么多快乐的人”。她相信他也有过真心，只是“这单薄的爱实在不足以支撑我们去走以后的路”。话是这样讲，但夜深人静，回忆起这段感情，她内心的创伤还是在汩汩流血，她甚至怀疑，这伤口永远也不会好了。“我再也不想去这样的付出了，再也不能对一个人有这样的温柔和耐心了，也不想和谁在一起了。我感觉，自己封起了自己的心。”她说，“晚睡姐姐，我该怎么做呢？以后，我还可以拥有一份好好的正常的爱情吗？我怕我的热情和耐心已经全部消耗在了这一个人身上。”我替小静鸣不平，世间有那么多好男孩，她却偏偏成为了“把妹达人”的牺牲品。他是她躲不过去的劫难，因为他学会了一切“搭讪（认识）、吸引、建立舒适和联系、到发生亲密关系”的技巧，经过反复的实践，演练，“3个认真的女朋友，8个以上的逢场作戏”，这样的战绩想拿下小静这样的爱情菜鸟，简直是易如反掌。他是这个浮世中某一类男人的典型——对情爱是完全的狩猎和掠夺。他并不懂得尊重女性，即使在获取女性欢心的过程中有过那些温情——体贴、关心、耐心——也不过是狩猎的手段，他心中的某一部分一直是冰冷而僵硬。小静以为是爱，他只当她是个盘点战况的数字。如毕加索。他一生中与他的画同样精彩的是他的情史，“混迹在蒙马特的模特，俄国舞团的女星、17岁的金发少女、先锋的女摄影师、年轻的画家……这些女子在他前面，付出感情、身体、之后是交付意志，直到一无所有。”女人是毕加索永远干涸的艺术源泉，他将她们7位中的6位凝固在了他的作品中，就像生前与她们纠缠不休，死后她们也永远的和他联系在一起。不想总抱怨男人，事实上男人也可以是受害者。网上流传甚久的小三攻略就详细收录了如何抢到别人男朋友的种种手段，细致入微，堪称手把手的教，“热情地要到他的手机和qq或者msn，经常发信息问他点问题制造接触机会······问的越白痴越好，越白痴他越觉得自己厉害，男人就喜欢这个。顺便请他吃饭，增进感情······”据说有人逐条实践过，成功了。事后，即使男人知道了，也不觉得是上当受骗了，而是“真爱”。爱情可以被这样批量生产，尤其是对于只看手段的人来说，对照着这些条条框框，就能模仿出爱情的赝品，且是高仿，看起来和真的差不多，除了那里面的“核”——真心。爱情可以有技巧，但爱情不能只是技巧。当爱情变成技巧，它便不会公平。一头是诱捕的笼子，一头是毫无察觉的猎物。真的赢了，游戏的乐趣基本消失殆尽，所以很多人都是在追求的时候最有兴致，只满足于收获一串数字。得手之后，马上热情大减。不要过度追求爱情的细节，是避免得到流水线上下来的批量爱情的关键。没有完美的爱情，不要幻想得到一个你无需开口就什么 # 老实男人也没有感情安全阀


在文章、黄海波等明星“出事”之后，天涯论坛上有网友发帖，说别伤心，好男人还剩一个呢，比如陈赫。然后呢，大家都知道了，陈赫也在不久被曝婚内出轨。那位实在的网友将自己的帖子捞出来，主动打脸，才让一众网友发现这个大笑话。现在，这个笑话又轮到了凭借真人秀《极限挑战》人气飙升的“大松鼠”王迅，这个一直不太有名气的演员凭借自己在节目中表现出来的憨厚和宽容赢得了一票女粉丝的心，被看成是新一代好男人的标准，结果被爆料曾婚内出轨，他的微博小号、前妻的微博、现任妻子的微博也被网友一一扒出。王迅的微博小号名字为“徐杨的幸福时光”，签名是“媳妇儿，你是上帝赐予我最好的礼物！”除了被删掉的一些，现在仅存的微博几乎都是在秀恩爱，而且很多自拍和照片，简直是铁证如山，不容辩驳。根据微博上透露出的时间线，2013年3月，王迅和现任妻子约会，2013年8月，王迅和发妻离婚，2013年10月，现任晒出与王迅的婚纱合影，2013年12月2日，王迅小号证实领证，2014年7月，两人的儿子出生。有人说王迅出轨，代表他根本就不是什么老实人，但看完他的微博，不得不说，他还真的就是一个老实人。在这个被网友誉为“史上最实在的小号”上，他毫不掩饰，实心实意地赞美自己的这段新感情和新恋人，诸如：“工作回来，看到媳妇儿的爱心水果酸奶，太感动了！谢谢我的好媳妇儿！爱你！上辈子我是做了多少善事，才修来今生的福气！谢谢你媳妇儿！桥底辣蟹，两个菜一千三百多，屌丝为了女神豁出去了！为了面子，不怕损失票子！不过真的有点儿贵啊！香港精英都不来消费，内地屌丝赢了！媳妇儿又给我做私房菜了！你们就羡慕吧！不好意思，哥们儿先享受了！媳妇儿第一次做黑芝麻糖，居然有百年老店的手艺！老公太佩服你了！太爱你了！”相比较那些鸡贼的，即使在小号上炫恩爱也遮遮掩掩，时刻删除证据的男人，王迅的微博小号至今还保留着这些东西，估计是没舍得删掉，只有笨男人才会这个时候还在乎这些虚幻的东西。不老实的男人出轨什么样？就像文章那样，出事之后，第一个和小三撇清，溜回家里继续扮演好丈夫好父亲，这是权衡利弊之后的选择，从来都不是感情上的谁输谁赢。或者像陶喆那样，百般抵赖，把责任全都推到对方身上。最容易一脑袋扎到婚外情中出不来的反而是那些看起来认真、实在的人。油滑的男人出轨是当自己出来玩，并不动真心，老实男人出轨才真的是身、心皆出，心甘情愿跳到这个游戏中不能自拔。王迅之前并无劣迹，相反，他虽然在演艺圈是近几年才暂露头角，但在四川省内还是小有名气，顶有四川省青年联合会军队政法界秘书长、成都市青年联合会委员、四川省曲艺家协会理事等头衔，早年就获得过五个一工程奖、群星奖、第一届巴蜀十大笑星等荣誉，还是很风光的。他对事业执著，对朋友重情重义，2012年，黄渤成立个人工作室时，第一个签约的艺人就是王迅，之后只要有好的机会，黄渤都不遗余力地帮助王迅争取，还在专业上帮他提高。包括在《极限挑战》中，很多观众都觉着黄渤孙红雷他们太欺负老实人，可王迅自己说，“他们欺负是在给我戏份，让观众注意我。”正是这种推举，才有了王迅通过节目迅速窜起的高人气。试想，像黄渤这样已经混成了娱乐圈人精的人，能提携一个品行很差劲的人吗？   在前一段婚姻中，王迅也并非就是一个糟糕的丈夫。2009年，在结婚10年之后，王迅曾补办了一场婚礼，并为前妻魏臻送上一个特殊的礼物，一件椎间盘患者使用的腰带，因为前妻有很重的椎间盘突出。说起自己和妻子10年的感情时，王迅不禁潸然泪下：“我平时拍戏忙，在家时间少，我搬了7次家，每一次都是老婆一个人操劳。”说这话的时候，前妻一直紧靠他的肩膀。王迅转身抱住老婆，轻轻说了句“谢谢你……”感情有时候难分真假，只有此时此刻，彼时彼地。那一刻的眼泪，也不会是假的，只是，难抵诱惑。一转眼，就换一场戏上演，患难夫妻中的好丈夫变成了老夫少妻中的人夫。这很奇怪吗？不，不奇怪，因为老实，并不等于有道德，更不等于品行好。有些看起来老实的男人为何也会不靠谱，是因为这种老实以自我压抑为代价，在没有机会唤醒人性中的贪心和欲望之前，很多原始的恶尚且沉睡着而已。王迅是个军人，而且是那种被部队塑造得特别有板有眼、守规矩的军人，在进入演艺圈之前，他没有去过酒吧，因为那在他看来是属于“灯红酒绿的场所”，藏污纳垢，必须远离。拍《疯狂的石头》时，他与黄渤初相识，黄渤在剧中扮演小偷黑皮，那个长相加上对角色的投入，在戏外黄渤也是一副地痞相，这让王迅很不舒服，“我是穿着军装去的，跟他们待一块会本能的感觉到不舒服，因为他们这种人是我们平时的斗争对象，真的看不惯，拍那个戏时我们俩几乎没怎么说话。”在这个时代，还能说出“斗争对象”这种话，可想而知，王迅对这个社会的认知是何等的滞后和狭隘。对于我个人来说，我习惯同一切看起来特别正义凛然的人保持距离，因为这种看似特别正直的人，执守的并非是信仰，相反，往往是不宽容、偏执和无知。见识得太少，才会画地为牢，将所有和自己不同的人都打入异类的范畴。而真正的仁人君子恰恰是宽容的，愿意了解和接纳各种不同的生存状态，正如真正的绅士从来都尊重一切贩夫走卒，不会看低那些比自己弱小的人。在王迅身上，恰恰是如此，他的正直和厚道都是一种危险的、不平衡的品质。在一个封闭的小环境之内，欲望被折叠成在一个最不引人注意的角落，他保守、戒备一切可能勾起自己欲望的东西，这种逃避，让他处于一种被压抑的状态。被压抑的人性，是时刻都在寻找突破口的。一旦王迅遇到新的诱惑和刺激，全部理智和节操都立刻瓦解。在现实中，出轨了就回家要求离婚的男人，多半都是像王迅这样老实的、恋爱经验少、认死理的人，他们以自己的绝情和放纵让曾经的爱人大跌眼镜。这种情况就好像一个穷人，没有钱，会连想象力都会受到抑制，只知道皇上的生活是东宫娘娘烙大饼，西宫娘娘裹大葱。等有一天他有了钱，会加倍地疯狂地不理性地消费，这是一种自我的补偿。自古以来，中国人就强调以德修身，但对道德的过高要求，反而加剧了人性的压抑。比如某市出台规定，要求公务员“不得观看、收藏、传播黄色影视和书刊，拨打色情电话”，反腐关口前移，深入到私生活领域，想把广大人民公仆打造成一群高尚的人、纯粹的人、脱离了低级趣味的人。他们混淆了私生活和人的综合素质、品行之间的关系，觉得道德高尚的人必须完全纯洁，不看不健康的东西，不说黄段子，非礼勿视，夫妻生活永远只会一种姿势。若看了A片那么“恶心”的东西，还要研读、学习、模仿，必然品行有亏，内心肮脏。中国人自古就对性事讳莫如深，发明了很多委婉含蓄的词汇来代替这件事，云雨、肌肤之亲、鱼水之欢、翻云覆雨等等。但嘴上不说，不等于心里不想，相反，欲望被压抑之后，往往爆发得更加猛烈，变成了超越生理需求的一种执念。那些卫道士们之所以热衷于对别人家的性事那么感兴趣，都是因为内心压抑，自己得不到的也别想让别人得到。食色性也，性和吃饭一样，是人的正常需求。法国人说最完美的妻子是“客厅里的贵妇、卧室里的荡妇、起居室里的仆妇”，说的就是人该在不同的角色中做不同的事情。总惦记着自己是“人”，那会活得毫无趣味，总停留在“动物”状态，那真成衣冠禽兽了。只有在人与动物之间自由切换，才能活得充实圆满，扛得起各种刺激和诱惑。太多出轨的人都输在了只给身体穿上了衣服，但心还没有穿上，裸着呢。他们经不起考验，当外界的诱惑力出现，往往全线崩溃，脱掉了道德的底裤，一路裸奔到底。控制自己的欲望，那需要更高层次的修养，绝不仅仅是义正言辞的拒绝和逃避就可以办到。所以现实生活中，反而是有过见识、历经一些沧桑的人更能坚守信仰。现在很多女人喜欢选择老实人，想要为自己的感情加一个安全阀。但老实人频频出现各种不靠谱的传说，早就证明，在感情中，所谓的老实已经不是加分项了。不要追求一劳永逸的那种感情，男人和女人都一样，都是在尘世中浮沉，寻找那个真实的自己。这个过程中，会有犯错和迷失，谁都难以幸免。还不如选择一个自己真心相爱的，一起在人生路上摸索，至于老实不老实，可靠不可靠，不走到最后，哪里会知道？查看历史消息可收看往期文章。阅读原文可以进入我的新书《你配得起更好》京东购买界面。

# 这并不算爱情的胜利

# 别敷衍你所在乎的生活


讲述： 晚睡姐，我一直很喜欢你文章，也是你的粉丝，结婚一年了，和老公吵了无数次架，偶尔我们还会动手打架，他打过我几次，这一年里我对婚姻很疲倦，当初和老公结婚是有点任性的，因为我刚好结束了两年的恋情，其实打心底我还是爱前任多点，结婚后我和前任也闹得不欢而散，有时候我觉得我很失败，我现在经常会想离婚，我老公容易对我动手，但是，结婚才一年很多客观原因也让我无法下定决心，我对我老公也还是有感情的，前任也马上要结婚了，我经常还会多想，我不知道自己是不是心里有问题，我很无奈，希望晚睡姐姐可以在百忙之中给我回复，非常感谢晚睡姐！ 我有位朋友刚上班的时候闹过一件糗事，同事大姐两口子吵架外加动手，大姐吃亏了，身上青一块紫一块，到单位哭诉，引来一众人等的同情。女人嘛，那骂男人的时候，肯定什么话都说了啊，“他竟然敢打我，我要和他离婚，呜呜呜，我再也不能和他过了，呜呜呜……”我朋友年轻气盛，直接领大姐回家找男人算账，说要代表大姐离婚，要告他动手打人。男人不否认自己动了手，“但是她，你问问她，都干了些什么？”男人也好一顿抱怨，朋友一听居然也挺委屈，然后大姐反过来抱怨朋友挑事，说你走吧，我这日子还得继续过。我朋友铩羽而归。回来和我说：“以后再也不能干这种事了，太丢人，都不了解情况就瞎主持正义。”我讲这个故事，绝对不是要说有的女人就是应该打，打人肯定是不对的，女人再不堪，男人有离婚的权利，但是没有动手的权利。我只是想说，很多不幸福的，闹闹吵吵的婚姻，都另有内情。人人都说自己的道理，却忘了人与人之间的行为是互相激发的。女人抱怨自己被生活亏待的时候，很少会反思自己都为此做了些什么，有时候她们只想有人给主持公道，但外人的主持公道是多么简单和肤浅啊，就像我朋友，无法理解为何不离婚，可不离婚就是有不离婚的道理。比如你，虽然承认男人动手，但是也隐约透露出你们还是有感情的，也承认自己对婚姻不够专注，还惦记着之前的男友，那么造成今天的样子绝对不是一个人的问题，你大概自己也认识到了。一个每天都在想着离婚的人，能对婚姻有多少热情和耐心，就可想而知了。婚姻也是是有生命力的东西，你敷衍它，它也会敷衍你，你对它不专心，三心二意，它也会对你不忠。别敷衍你所在乎的生活。不仅是生活，别的那些你在乎的东西，友谊、亲情、事业、自我，都不可以敷衍，都要认认真去对待，认认真去付出。我被很多人问过，我不幸福怎么办？这没有一个标准答案，而只接近标准答案的就是先问问自己能为争取幸福做点什么。幸福中哪有那么多侥幸，有的都是实打实的经营和付出，如小燕衔泥一点一滴的垒一个幸福的小窝。低头看看自己，有否对婚姻尽力，有否真正的尊重自己的选择和感情。如果答案是肯定的，那么你尽可以离开，如果是否定的，那还是好好的在这场婚姻中扎下根来，心无旁骛做好一个妻子的角色，或许，你会得到一个全然不同的生活。查看历史消息可收看往期文章。阅读原文可以进入我的新书《你配得起更好》京东购买界面。

# 那些说“不想结婚”的撩男，真实原因竟然是这个


问：潘老师，我跟男朋友是同事介绍认识的，之前交往了半年，我问他以后我们会怎么样？他说很难走到结婚那一步，因为各方面差异太大，父母也建议他找本地的，我很失望了提出了分手。当时他很不舍，但我强制性的断了联系，过了半年我还是放不下他，就加回了他微信。刚开始没有聊天，但他会点赞我朋友圈，也会去我家楼下以发朋友圈的形式告诉我，我就忍不住发消息给他，他说很想我，我们就重新在一起了。可在一起两个月时间，他都没有计划说两个人要重新开始好好发展，我就再次问他怎么想的，他说结局是一样的。我不理解，他明明很喜欢我，为什么就不想跟我有未来？潘幸知：一个不能跟你谈未来的男人，不要跟他在一起，除非你只是寂寞找一个伴，作为“短期选择”。两性求偶策略可以分为短择和长择两种。短择就是我根本不在乎我们之间有没有未来，现下找不到更好的，那我们在一起先处着也不错。短择最明显的特点就是不会和你一起规划未来。你会发现他做的所有决定都与你无关，甚至你们分手与否也和他无关。他可能会做很多过分或者冒险的举动，因为他根本不在乎以后。他想要的，只是现在一段关系。长择就是我确实认真了，想要和你长久的走下去。虽然可能两个人也吵架，也闹分手，但是他做的所有决定都是和你有关的。他未来的规划里有你，也期盼着带你一个个去实现。长期的伴侣会把你的感受放在一个十分重要的位置。虽然长择不见得能走到最后，短择也可能在相处的过程中变为长择。但是一个明确和你说不合适结婚的男人一定是短择。“不适合结婚”是什么意思呢，就是反正我们不可能在一起，我也告诉你了。你要是还愿意和我在一起那是你的选择，万一分手了你也不要怪我。这是你主动选择的。说白了就是“不主动、不拒绝、不负责”。这个男生是一个很精致的利己主义者，他知道你们走不到结婚那一步。你说他父母建议他找本地的，但你怎么知道他父母建议他找本地的，还不是因为他告诉你，父母建议他找本地的。他就是这样的一个态度，这也是他未来的借口。然后呢，他还是愿意跟你在一起。毕竟现下也找不到合适的长择对象，那么对于他来说，有一段短期的“加餐”也不错。姑娘啊，都已经半年了他都没有回头来找你，而是任凭半年之后你再去找他。说明他的内心很坚定你是“可有可无”的短期选择啊！如果你觉得自己真的喜欢他，愿意飞蛾扑火，哪怕100天之后，或者说两年之后分手也没有关系。不在乎天长地久，只在乎曾经拥有。那其实你们在一起也是合适的。如果你想要一个更明确的未来，想要找到在余生知冷知热对你好的人。那么你要清楚，这个男生显然是不合适的。所以你要自己决定，你到底想要怎样的一段关系？他明确说了，你们不适合结婚。其实就是他可以明目张胆的和“家里介绍”的对象约会，可以不顾你的感受做决定，哪怕有一天他和别人在一起了，也是家里人的决定。如果哪一天他真的找到“适合结婚”的另一半，你们之间的“爱情”也就结束了。

# “圆满”的婚姻里，藏着令人不寒而栗的风险

# 被怒骂“惯三”，从“啤酒妹”到双料视后，她是TVB最后一名黄金花旦，上位之路有点牛


最近，由吴彤导演的艺训真人秀《无限超越班》频频上热搜。因为请来了诸多老牌港星，所以“情怀”无疑成了《无限超越班》的最大噱头。而在此期间，佘诗曼更是成了一个特别的存在。在翻拍的7个IP作品中，佘诗曼参与的作品就有3部。《金枝欲孽》、《宫心计》、《使徒行者》。这三部在港剧迷心中，可是占下了不少的分量。而在《无限超越班》中，佘诗曼更是以当年《金枝欲孽》中尔淳的造型示人。让网友无不感叹：“岁月不败美人，跟当年一模一样。”不得不说，佘诗曼虽然只是短短演了那么十几秒，都能让人一下子梦回《金枝欲孽》。作为陪着无数人长大的佘诗曼，虽然相貌在娱乐圈中并不算出众，但她却依旧能在圈中杀出了一条属于自己的路。从出道时被群嘲“鸡仔声”，到如今凭借演技拿下视后，成为TVB时代最后一个黄金花旦。她的逆袭之路，正如古诗所写“不经一番寒彻骨，哪得梅花扑鼻香”。01 在娱乐圈，佘诗曼是出了名的情商高。但正如马云所说，高情商往往来自于生活所受的苦。1975年，佘诗曼出生在香港一个普通家庭，虽然算不上大富大贵，但起码也算小康。生活美满，父母万般疼爱，佘诗曼5岁前的童年过得十分快乐。一切本该如常进行，但没想到一场车祸带走了佘诗曼的父亲。一下子，所有都变了。作为长女的佘诗曼，非常心疼母亲，她跟母亲说：妈妈，我从今天开始陪你睡。为了养家，母亲不得不更卖力地工作，每天都是早出晚归。而家里的大小家务事、照顾弟弟妹妹，则落在了佘诗曼的身上。虽然年纪小，但她却依旧能把家里打理得井井有条。在母亲和佘诗曼的共同努力下，她的生活才重新步上了正轨。1993年，18岁的佘诗曼前往瑞士，读饭店管理专业。在那里，她遇到了自己的初恋，一个来自台湾的富家子弟。毕业后，佘诗曼跟男友回到了台湾。在台湾那段日子，佘诗曼提前过上了富太的生活，无忧无虑。但日子久了，佘诗曼反倒对这种生活感到无趣，甚至起了想要逃离的心，她觉得自己不适合做阔太，反倒更适合打拼。于是，她忍痛和男友分手。回到香港，刚满22岁的佘诗曼心心念念想找到五星级酒店的工作。但奈何那个时候正处年末，大部分酒店都不招人了。为了生计，佘诗曼不惜去酒吧卖啤酒。后来，在妈妈的推荐下，她报名参加香港小姐比赛。在那个时候，港姐比赛可谓是神仙打架，个个都是美人。相比之下，佘诗曼的样貌则显得平平无奇了。但幸好，她的高情商为她扳回了一局。在问答环节，主持人问她，“如果香港首富是你的邻居，你要劝他搬家，你会怎么做呢？”在场的人听到这个问题都下意识愣了愣。没想到，佘诗曼十分从容淡定，笑着回答，“如果我成为了香港小姐，会有很多粉丝来我家，那他会受到骚扰，可能会让他有麻烦咯。”满堂喝彩。凭借着从容不迫的表现，端庄大方的形象和高情商，她成为了当年选美的黑马，一路挺进三强，并拿下了季军。而当年的冠军是翁嘉穗，亚军是李明慧。按照老规矩，三人都被无线签下，成为力捧的对象。但谁都没想到，翁嘉穗和李明慧为了一个富商，争得你死我活，为了嫁入豪门，两人甚至明面搞针对，不惜与无线解约。无线气死了，于是立马解除和两人的合约，并将所有的资源通通给到了佘诗曼。当时的佘诗曼，因为这场无硝烟的竞争，成为了无限的“资源咖”。02 她参与的《雪山飞狐》，一来就担大旗，出演女主苗若兰。但可惜，这并没有给她带来什么好的影响，反倒因为表演过于僵硬、眼神没戏被观众群嘲。有人说她不会演戏，就是个花瓶，有人说她的声音像鸡仔声，有人甚至给她起外号“嗲精”。不得不说，当时香港观众也是狠，遇人也真敢往死里喷。佘诗曼一开始也被这些谩骂声弄到怀疑人生，萌发了退圈的心，但奈何合约在身。当时香港娱乐圈的导演，只认戏不认人。面对不会演戏的佘诗曼，有一位导演甚至气到将剧本怒摔在地上，并且用脏话不断问候佘诗曼的家人。但作为一个完全不懂戏的小白，佘诗曼知道自己不能哭，必须得赶紧学会演戏。于是，她拜前辈陈锦鸿为师。陈锦鸿对她来说，是一位良师，他不断地点醒佘诗曼，他说，台词不行，就天天大声读报纸，没演技，就每天看镜子酝酿入戏。佘诗曼坚持这么做了。果然演技得到了长进，嗓子也开了。2000年，佘诗曼迎来了她第一波的运气，搭档张智霖主演电视剧《十月初五的月光》。该剧一上映，便好平如潮，收视率节节攀升，刷新了香港电视剧的收视纪录。凭借这部剧，佘诗曼获得了观众的认可，并成为了当红女星。无线趁热打铁，让佘诗曼接连出演了《碧血剑》、《倚天屠龙记》、《帝女花》、《金枝欲孽》等佳作。每部剧都是口碑和收视双丰收。佘诗曼坐稳了TVB无线一姐的位置。后来，她还凭借《凤凰四重奏》成为双料视后。她演绎的每一个角色，无不让人印象深刻。就像《金枝欲孽》中的尔淳，心思缜密，冷静自若，妥妥一个腹黑大女主，但实际内心善良，爱而不得，却依旧甘愿为爱人而牺牲，她留下的那一滴泪，让人无不动容。在《凤凰四重奏》中，更是一人分饰四角，精准演绎出每一个角色不同的人物性格。她演的角色，真的没有一个是好演的，但她却依旧能将这个角色演绎得淋漓尽致。当然，每一个高光时刻，都离不开背后的付出。一年拍120多集的电视剧，每天工作18小时，3年只休了9天假，她硬生生将自己从一个人见人嘲的演技小白逼成了拿奖专业户。正如亦舒所说，做人凡事要静，静静地来，静静地去，静静地努力，静静地收获，切忌喧哗。而佘诗曼，就是这样淡如流水般，静静地抹平了曾经所有人对她的嘲讽。2011年，佘诗曼开始北上寻求更多发展，也拍下了不少作品。但跟很多北上的香港演员一样，她也遇到了事业的瓶颈。高不成低不就，资源上也并不充厚，为此，佘诗曼一度被港媒嘲笑为“北上发展最差的花旦”。TVB高层也竭力劝她回巢。但她也只是回来拍了一部《使徒行者》，顺带拿了个奖，便匆匆回到大陆。她不甘心，想趁着潮流再拼一把。终于，她等到了《延禧攻略》娴妃一角。而也正是这个角色，让佘诗曼成功打开大陆市场，事业达到新高度。她的成功，离不开她的实力，更离不开她的谦逊。在拍摄时，为了能更好入戏，佘诗曼还是选择用粤语念台词，但为了能让对手接戏，她还是很细心地会将最后一句台词转回普通话来念。而且，虽然演的是配角，她也没有轻视这个角色，自己的戏份拍完了，她也不回酒店，反倒是留在片场看其他人的表演。以谦逊地姿态，顽强地走过低谷，让娴妃一角焕发出主角的光彩。想必，只有佘诗曼能做到。03 然而，人红是非多，随着名声大涨，佘诗曼的绯闻更是被一一翻出。跟张智霖因为合作《十月初五的月光》后，被媒体曝出两人因戏生情。而此时张智霖已经和袁咏仪谈了9年的恋爱，为此，袁咏仪不禁大怒，跑到片场公开指着佘诗曼鼻子骂：“就算张智霖不想娶我，也不能你这个‘惯三’来炫耀。”当然，所谓的绯闻也只是港媒搞的鬼。多年后，袁咏仪在节目中向佘诗曼道歉，称自己以前不懂事，听信了媒体上的报导，误会了佘诗曼。的确，当年的佘诗曼早有心上人，不是张智霖，而是陈浩民。陈浩民作为港圈情场高手，轻而易举就将佘诗曼拿下了。虽然两人极力否认恋情，但没想到港媒还爆出了两人尺度很大的通话对话。看到对话的文字版，所有人的都惊呆了，没想到看起来女神还有不为人知的一面。眼见事情越闹越大，陈浩民立马站出来怒批媒体没有下限，窃听隐私，而这也侧面坐实了两人的恋情。可惜，两人终究没走到一块，紧接着，佘诗曼就跟小4岁的吴卓羲传出了绯闻。佘诗曼更是送其价值近港币20万元的名表以及200万元的名车。于是，吴卓羲被港媒戴上了“软饭王”的名称。但自尊心超强的吴卓羲怎么能忍受呢，于是他便把自己的兄弟郑嘉颖介绍了给佘诗曼，自己趁机远离。而佘诗曼和郑嘉颖的确也互相看对了眼。但可惜，当时郑嘉颖太年轻，在情感上少了一份担当。两人被记者拍到约会，为了避嫌，郑嘉颖居然直接让佘诗曼下车，让她独自一人在大街上等车。事后，郑嘉颖居然还当众说“有上车就会有下车”，这句话让佘诗曼颜面扫地。这一举动彻底伤了佘诗曼的心，两人自此各奔东西。04而如今，已经47岁的佘诗曼，依旧是单身一人，被问及什么时候找男朋友的时候，她笑着说，我爱工作。虽然嘴上是这么说，但她并不是人们眼中的工作狂。翻开她的社交平台，你会发现，与其说她爱工作，倒不如说她爱当下自洽的人生。闲暇时间，她一直在不断充实自己，过自己喜欢的生活。学画画，还学得有模有样，虽然是业余爱好，但作品完全能拿得出手。几十年如一日的健身、练习瑜伽。因为参加了一档美食综艺，她学习做菜，在节目中拿下冠军，在生活上则开始了她佘师傅的做菜日常。除此之外，她还酷爱和媒体记者“玩车技”，为了等记者追上来，她会开的很慢，当记者追上来的时候，她猛踩油门，一溜烟不见人影了。在她的生活中，除了事业要求她必须是个成熟的大人，其它时间里，她就像个对任何事物都充满好奇的小孩，勇于学习和尝试，并且懂得为自己的生活增添乐趣。她活得热烈而自我，潇洒而又真实，完全没有所谓中年女星的迷茫与无措。正如她自己所说，聪明的人，有中年，没危机。点击下方，添加星标⭐️不再错过潘幸知

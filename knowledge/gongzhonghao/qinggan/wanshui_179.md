# 如何在庙小妖风大的地方生存


讲述： 晚睡姐姐，我想问你个问题，职场上，如何不被别人欺负，我们这边是驻场设计所，平时一共就三个人，我和我以前的同学，还有一个40多岁的同事。最近，我上班都是迟到个十分钟，以前我都是准点到的，最近天冷，而我同学每天迟到很久，周六还会迟到一个小时，但是我同事都没有说她。我同学很多事情不会做，她就基本不怎么做，本来我们这边人少，她不干活，就只有我来做，我和我同学工资还一样的。有时候加班熬出来，也不是很大问题也被同事说，或者他忘记了他修改了，我感觉很吃力不讨好的。我也尽自己的力量把工作做好。我不喜欢这样一直被消耗着，我想保护自己起来。中国有一种特别糟糕的职场现象，就是庙小妖风大。明明没几个人，工作也没啥技术含量，偏偏搞得跟宫斗似的，天天热衷于互相算计，脸上似火盆，背地里下绊子，不是排挤这个，就是诋毁那个。正所谓一条鱼腥了一锅汤，要形成这种恶劣的风气，不用人人如此，只需要几个市侩、势利的人搅合就行了。特别是某些机关事业单位，养着很多有背景的，比如某位领导的夫人，谁谁的小姨子，自觉有背景高人一等，谁都不放在眼里，那绝对能引领潮流。最后老实人靠边了，好人不趟这浑水，感觉蹦跶的全是这种人。如果不幸遇到了这种庙小妖风大的地方，该如何生存下去呢，听我几招：第一要注意不犯错误。什么鱼最难抓，就是鲶鱼，身上滑溜溜，没把柄，抓不住。环境已经很复杂，就不要以为浑水摸鱼是对的。要尽好自己的本分，这是在任何职场都必须恪守的原则。谁迟到早退都和你没关系，千万别攀比这种事，这是给别人提供炮弹的行为。用“你干嘛批评我，你咋不看他呢”这种逻辑来为自己辩护，什么用都没有，还暴露出了自己的幼稚和不成熟。第二要注意不搞小团体。不站队，也是职场的重要原则之一。都是同事，面上都应该一碗水端平，如果觉得自己加入某个小圈子就能抱团，获得某种庇护的话，那么风险也是显而易见的——小团体最容易被从内部攻破，而且也容易被外人视为整体的威胁。越是复杂的局面中，越适合明哲保身，不偏不倚，才是存身之道。第三要吃得起亏。中国有句老话，“来说是非者便是是非人。”事实上，很多觉得别人对自己不好的人，身上其实也都有自己的问题。在任何环境中，最讨人喜欢的，都是那些大方、爽快、有原则，又不拘泥的人。你必须能吃得起亏，小事上不计较，不过分玻璃心，才是事业和人际关系双丰收的保证。第四要牢记职责分工。一定要按照管理层次做事，谁是你领导你听谁的，不是你领导来指挥你不要听，你让他去找自己领导。千万不要随随便便的就接下不归自己管的工作，往往是吃力不讨好的结果，遇到个别心眼坏的，一旦出现了纰漏，还容易直接让你去顶雷。而且这种习惯要是形成了，以后谁都会来指挥你。当然要同事之间可以帮忙，但要明明确是帮忙，怎么做，做到什么程度都是对方承担的问题，你只负责处理，是人情，不是本分。第五是抓住把柄，一举反击。能吃亏，受得起委屈，也是在一定的范围之内，真要是被欺负狠了，也不能还是忍着，这不是大度，这是懦弱。但要反抗，不要在小事上，什么今天谁给了我一个白眼、昨天他说话语气不好这样鸡毛蒜皮的事情上闹，那显得特别小家子气，还掩盖了实际上的矛盾。要搞，就要彻底搞死，一定是抓住对方大的把柄，比如贪污、渎职或者重大过失，以及在某些事情上特别过分，拿到月球上说都是你有理的错误，大闹一场。不要怕闹翻脸，你这么做一次，下一次没有人敢再欺负你。总之还是一句话，人生当以不变应万变，没事的时候不惹事，事来的时候不怕事。阅读原文可以进入我的新书《你配得起更好》京东购买界面。点击右下角的往期文章可以进入文章分类链接

# 什么样的分手适合复合

打的旗号是要展现正宗摸金范儿，其实看完《寻龙诀》之后，发现和摸金也没什么关系，只是用了盗墓的噱头讲了一个关于放下过去的故事。当然，这个故事本身还是挺好的，值回票价，提前安利一下。没盗成墓，让鬼吹灯的真爱粉失望，这也怪不得他们，谁让在广电总局的大剪子威胁之下，不能盗墓不让成精不给有鬼呢。胡八一和王凯旋、Shirley杨回到美国准备金盆洗手，王凯旋无法说动胡八一重操旧业，只好自己私下里去接活，却意外发现，自己二十年前的战友丁思甜死去时候的彼岸花再现人间。他拼死要去趟这趟浑水，想要借机去查明丁思甜的死因，临走之前，给胡八一和Shirley杨留下了线索。算起来丁思甜只是王凯旋一厢情愿的暗恋对象，她真正的初恋情人是胡八一。在他们插队的蒙古包里，胡八一把一枚像章送给丁思甜，“你愿不愿意将我们的革命友谊再升华一下？”丁思甜扑闪着大眼睛，羞涩地问：“你想怎么升华？”还没等胡八一回答，便被外面大声朗诵诗歌，前来示爱的王凯旋打断。三个人的感情只限于此，没有得到机会更向前一步。他们跟着一群狂热的知识青年进入地下古墓探险，变故发生，只有胡八一和王凯旋逃出，丁思甜为了救他们两个人，牺牲了自己的生命。二十年来，沉重的愧疚始终纠缠在两个人心头。王凯旋是一直单身一个人，胡八一是游离在Shirley杨身边，逃避着她的爱情。这次王凯旋重回令战友们送命的古墓，就是想要履行自己承诺给小丁的，“我负责给你采一朵彼岸花”。王凯旋在古墓遇险，胡八一和Shirley杨赶到，希望他不要再执著于这件事。王凯旋做不到，他不肯走，“我王凯旋爱一个人，就是一辈子！”小说中的王凯旋就是一个其貌不扬的人，电影中黄渤扮演的王凯旋更是颜值低到爆表，和陈坤扮演的胡八一两张侧面同时出现在屏幕上，人家陈坤的侧面鼻子下巴的线条是黄金分割，他的好像是上帝算错了数据，哪哪都不符合审美比例。但黄渤演出了王凯旋对爱情的忠贞，可以令人直接忽视外表，被他的气势所打动，他知道自己永远都得不到小丁的爱，但是他不介意，“我随时都可以为小丁去死！”王凯旋说：“有好多个夜晚，我都不知道自己是睡着还是醒着，我睁开眼，就看见小丁的脸在我面前，问我：我的花呢？”白天的时候他是油嘴滑舌的王胖子，夜晚他就是一个背负着沉重心灵十字架的罪人，他不肯原谅自己，因为小丁以命相救，他却没有兑现自己的承诺。这场被动的盗墓，其实就是他们三个人心灵的历练场，他们重回旧地，与念念不忘的故人相见，只为了能放弃纠缠于心的执念。当爱情变成执念，已经失掉了开始时候的宗旨，令人深陷痛苦之中。关于如何放下，黄渤之前还有一部电影《心花路放》。黄渤扮演的耿浩因为离婚，变得愤世嫉俗，徐峥扮演的好基友郝义深信摆脱痛苦最简单的方法，就是开始一段新感情。他开车带着耿浩四处猎艳，最终却发现，没有什么能弥补爱情的损失和感情的创伤，靠着寻求外界的帮助来解脱自己只能陷入更深的绝境。真正的解脱是接受自己的过去，“阴影也是人生的一部分”，失败的婚姻也是自己的一部分，前妻也是自己人生的一部分，正如影片的导演宁浩所说：“放下是寻求你的过去，接纳你的过去。就像有人说过，有了事情就接受它，处理它，放下它。”《寻龙诀》里，刘晓庆扮演的导师肃然说：“做人，总是需要一点精神的。”是的，除了精神，生活，还是需要一点绝情的。不绝情一点，我们怎么活下去？多少曾经热烈炙热的情感，冷淡了、沉寂了，离我们远去；多少曾经朝夕相对，被视为阳光空气和水，不可或缺的人，要从生命中生长摘除。可你还得活着，还得没心没肺的笑，还得装作若无其事的行走，过好每一天的生活。你可以痛心疾首，可以永远封闭自己，再也不去爱上任何人，或者像王凯旋那样，强调“我爱一个人就是一辈子”，将自己变成铜墙铁壁，不许别的感情渗透进来。可这样就是对爱情的最好诠释和回报吗？失掉爱人，让王凯旋心存执念。重获爱情，却让胡八一开始变得柔软。胡八一内心也一直记着小丁，但他则不同于王凯旋以命相搏的疯狂，他想的是，“小丁牺牲了自己的生命让我活下来，可不是让我死在这里的。”Shirley杨也说，让小丁的死去变得更有价值的，就是他们好好活下去。人是多么现实的动物，承认这一点吧。我们顽强的生存着，善忘、冷血、迟钝是我们保护自己的手段。道是无情却有情，情到深处情转薄。总有一天人们会想明白，与其活在冷冰的怀念中，不如留住身边一个温暖的微笑。片尾，胡八一拿出一枚戒指，对Shirley杨说：“你愿不愿意将我们的革命友谊再升华一下？”这句话，曾经在二十年前的草原上，对着一个梳辫子的少女说过，现在闯过生死关头后，站在漂浮棺材中，对另外一个完全不同的女孩同样说出了这句话。往事落幕，爱情得到了重生，破碎的彼岸花上故人的光影逐渐黯淡，没有人可以被真正遗忘，但从此后，过去的归于过去，过去的事放在心里。就该好好活下去，即使有锥心的痛苦。一个人，活到能享受爱情的年纪，已经算得上过五关闯六将了， # 如何和多疑的控制狂相处


讲述： 晚睡姐姐：您好！有一个问题：一个男人开始怀疑他的妻子，这辈子就会一直怀疑，很难打消，可以这么说吗？我和我老公结婚快一年了。我俩虽然早早就知道彼此，但从正式接触到结婚，也就7个月，算闪婚了。现在过的并不幸福，就是因为我老公总怀疑我婚前生活，怀疑太乱。今天早上突然把我手机抢过去翻了一上午，说我做人不诚实，翻手机找线索。天地良心，我对他真没有二心。他总是怀疑我跟很多人都有问题。有一天这种情况会有改观，会好起来，他能完完全全相信我是好女人吗？电视剧《不要与陌生人谈话》非常深刻地刻画了安嘉和这样一个控制狂加暴力狂的形象。而安嘉和的转变，看似是由外界的契机带来——梅湘南对他隐瞒了自己被人强暴的经历，导致了安嘉和不再信任她——其实本质上还是因为他就是一个具有偏执型人格障碍的人，十分敏感多疑，而且固执己见，一旦自己对某些人或者事产生了某种怀疑，就会执著于此，耿耿于怀，并伴随着强烈的报复心。安嘉和是一个典型的控制狂，梅湘南也是一个典型的被感情蒙蔽了自己眼睛的女人。她和你一样，相信了安嘉和的解释、忏悔，相信自己只要多给对方一些温暖，多注意一下自己的行为，就能让对方相信自己。这是正常人的思维，如果对方怀疑自己，那么一定是由于某些证据促进了那些怀疑，只要消灭这些证据，疑虑就会打消。对吧，正常人都是这样的。但问题是，有的人不是正常人，一个男人怎么可能会轻易怀疑自己的女人和很多男人都有问题呢？这不符合常理，对不对？那些偏执的人会忽视和自己想法不符的一切客观证据，只凭借着自己的逻辑作出判断，所以对于这样的人，摆事实讲道理，比如，“你看我平时都不和男人说话，我怎么可能出轨呢”这种话，是根本安付不了他们的。而且越是执著的努力证明自己清白，反而会引起对方的另一种怀疑，“你心里没鬼你会这么顺从我吗？”这样的人常常不信任任何人，不是不想相信，是不能相信，无法相信。他们没有信任别人的力量，因为他们就生活在一个阴暗的，别人都会欺骗自己、伤害自己的世界中，他们的怀疑是屈服于内心的恐惧和焦虑，而对世界投影出的一种行为。你对他们老老实实的顺从，不碰触他们的逆鳞，或许能安慰到一部分，但如果他们自己不懂得面对自己的人格障碍，他们就不可能从根本上恢复对伴侣的信任。和控制狂相处有几种技巧，可以适当避免一些伤害：一是要知道他怀疑、不信任的根源是他内心的焦虑，是病态的性格，基于这样的认识，可以减少自己不被信任的痛苦。二是要了解他在担心什么，搞明白他的思维逻辑，找到触发他的敏感点，然后在适当的妥协下达成一个双方都能接受的协议。三是要尽量营造一个和谐温暖的环境，有些内心焦虑的人长期置于良好的人际关系中，会改善自己敏感多疑的性格，逐渐恢复理解他人和信任他人的能力。四是要利用他真正信任的人来进行说服教育，即使是控制狂的世界中也有自己信赖的人，他们说的话他更容易听得进去。五是要尽量减少变化对他们的刺激，比如说好了做什么事，就要按照计划来，不要很随意地变化来变化期去，因为打乱计划就是滋生控制狂们不安的最好土壤。六是要有自己的底线，不要无限制的让步，可以反其道而行之，他怎么要求你，你就怎么要求他，要让对方知道自己得行为是很不尊重人的，很不叫人舒服的。和一个控制狂在一起生活会很不容易，因为他们并不知道自己是控制狂。他们总是认为自己有足够的理由这么想，如果你屈服，这种控制可能无限度的。所以一旦发现你改变不了对方，只能离开。否则他们会不断在语言上贬低你、精神上羞辱你，直到你认为自己一无是处，精神萎顿，情感消极。阅读原文可以进入我的新书《你配得起更好》京东购买界面。点击右下角的往期文章可以进入文章分类链接

# 坚持美好，是你和神之间的事情


讲述:姐姐，我家条件不是很好，我每年都申请助学金。成绩勉强还可以，但是感觉已经申请了一个了，就没再申请奖学金，想着把机会留给其他需要的。但是，我发现我们班好多家庭不贫困的同学都申请了助学金，同时还申请奖学金，我两个室友都是这样。。。现在我心里还挺不平衡的，一方面觉得她们这样做挺不对的，一方面觉得自己太傻。因为看到她们拿到1000块奖学金，而自己放弃了机会什么都没有，何况我比她们更加需要这个钱，还挺闹心的。。。这样为了钱不择手段，真的好吗？我现在已经不知道自己这样单纯是不是一件好事了。。。从小被教育做人要善良真诚，但是慢慢发现机会属于精明敢于打破所谓原则的人。那我们教育那些品德又有什么意义呢？ 前几天儿子回家闷闷不乐，我问他怎么了，他说一位好朋友批评他太大方，对同学总是太好，所以有人喜欢占他的便宜。我说你自己是怎么认为的呢，他说我没觉得谁占我便宜了，我只是对谁都挺好。我说那就继续做自己好了，朋友的话或许是在帮你打抱不平，但即使有人喜欢占便宜，你可以拒绝和他们继续做朋友，只是不要因为有人不珍惜自己的好，就改变自己人生的信条。慷慨、大方、真诚、坦荡，都是一些美好的品质，我教育他拥有这些品质，并不是为了有一天能拿这些品质来获利，而是我认为这才是人生终极的追求目标。我希望他能做一个干净、友善，永远都拥有赤子之心的人，无论身处何地。就像小宋佳曾经说过：“我不能让随便什么人就逼出我的负能量来。”有些人并不相信这些东西，人和人都有不同的价值取向。有的家庭教育就是不能吃亏，一定要占便宜、投机取巧。是的，很多人都是如此，和社会相比，学校还是比较干净的，等有一天你走到社会上，你就看到更多人在利益面前的表演，那真是丑态百出，无所不用其极。要失望吗？不，不要。苏珊·桑格塔说过：“一个人若是对堕落感到吃惊，见到一些人可以对另一些人施加令人惊悚、有计划地暴行的证据，就感到幻灭，只能说明他在道德和心理上还不是成年人。”一个真正成熟的成年人需要对不堪有耐受力，即使看过最坏的人性，却依然对美好有向往。每个人都要在这个世界寻找自己最终停留的位置。有的人勤勤恳恳，有的人偷鸡摸狗，有的人认真做事，有的人溜须拍马，鸡有鸡道、狗有狗道、人有人道、仙有仙道。走捷径的人走得快，踏实的人走得长，各有归所。我们所说的上天是公平的，是在一个大的领域所权衡的公平，在小范围，可能会出现好人吃亏，坏人得势的结论。但没关系，人间正道是沧桑，慢慢走，走到最后，总是能见到分晓。那些耐不住时间考验的人，早晚会暴露出真实的自己。“有的人可以在所有时间欺骗一部分人，也有的人可以在某一段时间欺骗所有人，可就是没有人可以在所有时间欺骗所有人。”特蕾莎修女有这样一段语录：“即使你是友善的，人们还是会说你自私和动机不良，不管怎么样，你还是要友善。“当你功成名就，你会有虚假的朋友和真实的敌人，不管怎样，你还是要成功。“你今天做的善事，人们往往明天就会忘记，不管怎样，你还是要做善事。“说到底，这是你和神之间的事，而绝不是你和他人之间的事。”特蕾莎修女把一切都献给了穷人、病人、孤儿、孤独者、无家可归者和垂死临终者。她从12岁起，直到87岁去世，从来不为自己、而只为受苦受难的人活着。 我们学不来这样的伟大，我们只要确保自己能坚持美好，在心中留有一个圣地，为了自己所相信的信念前进，就已经是对这个世界最大的贡献了。即使我们是无神论者，我们也有自己的神，就是心头的善与美。坚持美好，在没有人看到的角落，依然恪守自己的原则，才是真信仰。阅读原文可以进入我的新书《你配得起更好》京东购买界面。点击右下角的往期文章可以进入文章分类链接

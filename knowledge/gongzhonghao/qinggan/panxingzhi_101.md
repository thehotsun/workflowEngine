# 我41岁，经历2次生死，想告诉你这4个箴言

第一次触及到死亡的结果。上高中那会儿，有一个男生常说长大了要开飞机，可惜没到毕业，他患重疾离开。那时我们一帮十六七八的少年，隐约嗅到了失去的悲伤。第一次真正流泪，是在离开18个月后，第一次从德国回家，打开门兴奋地呼唤着猫的名字，空荡荡的过道。母亲说，猫走了，怕你难过，没敢告诉你。眼泪瞬间决堤，我忽然意识到，死亡可以如此轻而易举地夺走16年的陪伴。从那日起，我开始抵触失去的恐惧。03年，奶奶离世，未能送行，我在德国，一夜宿醉，赔上一段感情；07年，爷爷生病，陪伴我走了一段回家的路后，去世了。内心深处，我开始抵触，希望时间不再前进。15年外婆因手术意外昏迷不醒，基本只剩身体体征，一年后，外婆一直昏迷，大家终于决定让她走。那以后，我以为我开始懂得与死亡和解，因为我渐渐意识到它并非面目狰狞，它的出现让我开始思考生的真正意义。2018年，与自己和解的我，开始专注新的项目。同年4月，父亲忽然住院，查出胸腔有积液，诊断猜测可能是肺癌。入驻华西医院深入检查的那段时间，我天天祈祷一切会好，像个疯子一样在13层楼里上下跑动，心里揣着恐惧，却要面容和悦，不仅要宽父亲的心，更要让母亲有所依靠。这一次，我开始以我的视角正视死亡，它不仅仅是人们恐惧的负面，还有积极向善的一面，为我们自己，也为那些值得我们付出的人。2个月后，父亲走了，没有来得及交代任何身后事。来不及悲伤，大脑高速运作接下来的所有安排，觉得错失余生的同时，也庆幸昨晚还能让他再喝一碗我熬的汤。自始至终，我都未告诉他真正的病因，怕他难过失落，即使我知道他是个特别乐观却总是内敛悲伤的人。家中多为信佛的善男子善女人，父亲去世后，在安静的诵佛声中，我发现悲伤到最深的地方竟是满满的爱与平静，正因拿得起死，才能放得下生，不再执着生死得失，才能更全力以赴地生活付出，更懂得做人做事无我利他的可贵。2016年，我失去了事业，2018年，我失去了我的父亲。这两年所有的经历下来，让我对自己的年龄有一种深深的恐惧感，让我深深的感觉到了，在面临快40这个不惑，既不能上，又不能下的这种焦灼感。非常有意思的是，我在2018年，在我40岁生日这一天之后，我对着镜子说的第一句话是说，从今天开始41岁的日子我们一定要好好过，其实在那一天的时候，我突然意识到：我在17年和18年，这两年其实对我自己的实际年龄，我是非常模糊的，而且对这个年龄有一种恐惧感。之后，我做到了5件事情。第一件事情是和自己和解；第二件事情我觉得是和生死和解；第三件事情我觉得是与成长和解；第四件事情是与情感和解；第五件事情是我觉得我做到了去更多地信任自己和质疑普世观，从而在其中找到真正适合自己生存的方法。 04住在舒适区的你，虽生犹死我们生下来是一个点，被成长拉成一条线，又在社会里演戏，变成三角形，正方形，终于有一天变成圆，又恰在心里多了一点方正，与人有同理心，与己有原则，无疑是活出了自己喜欢的模样找到了和他人最舒服的距离。有一年，跟一个哥们儿在初春去爬阿尔卑斯山，偏偏走错方向，陷入一片皑皑白雪的滑雪地段，继续走似乎离峰顶不远了，两个人心想咬咬牙上吧，走了三分之一，才发现路滑道窄，必须手脚并用才可以前行，在寒冷中渐渐感到体力不支。哥们儿吃力地爬上最后的征程，消失在我的视线里，我开始感到害怕，人一旦在自己不可控的环境中就会慌乱，我停在原地休息把注意力集中在呼吸上，一点一点地往上挪，不停地对自己说，不要急，先走好脚下的这一步，大约过了半个多小时，我看见一块露在雪中的水泥地，踏上去，再回头，发现自己已经登顶。我趴在栏杆上大口地喘气，膝盖微微颤抖，极目远望，阿尔卑斯山脉峰峦跌宕，此刻的内心有些后怕，但更多的是一种喜悦。我知道，这是我人生又一次走出“舒适区”的收获。我喜欢徒步旅行，喜欢登山，喜欢骑行，它们带给我许多有关生命的启示。假如人生只能活到60岁，我将怎样度过余下的这段时间，我想我会时刻尝试地走出生命里的“舒适区”，这个世上还有很多好玩好看的事，活在每一刻，不观后不思前，足足把每个当下活到真实透彻。2000年，我只身一人前往德国，刚学德语时，我做的第一件事，就是搬去与德国人合租，在社会生活里学习语言和认识这个国家。学德语时，我抛掉了我们固有学习英语时的方法，直接去听说写，去建立以中式的思维去表达中文，以德式的思维去讲述德文的能力，这种不断在两种语言和思维模式间的切换，后来慢慢成为自己独特的跨文化沟通的能力。走过岁月中，有一半的时间，都与德国有关，双元文化已让我形成特有的对世界的思考和解读方式。我们这个社会，大部分都习惯并喜欢用自己的认知去绑架别人。其实我从未给自己做过任何有关文化的标签，我的认知里，我始终是那个自己，用自己的方式吸纳、筛选、思考和解读不同的文化，并且尊重它们在不同维度上的存在价值。每个人生存的环境和文化是靠自己改变的。我在试错和实践中，慢慢地建立了一套“颖模式”，在这个跨文化体系里，没有国籍的区别，没有对错好坏的二分思想，它只是尽可能完整地和人们一起去探索事物的不同视角，尽可能地触及事物的本相。这是我又一次尝试走出“舒适区”的收获，完成对自己的拓展。其间也感谢我的父母，他们开明放养的方式，给予我最温暖的爱和理解。这一生，很多时候只有走出“舒适区”后，我们方可明心见性，成就自我。走出舒适区，很大程度上意味着着孤独，但它并不可怕，其实仔细想想，好多美好的事情都是在一个人的时候发生的。走出“舒适区”意味着，我们终于停下来，不再用二分的方式看待世界，开始看到生命还有很多可以塑造的可能。生命的意义是真正地做自己，所有的经历都是修行，而修行其实是换一种心态来看待世界人生。我一直以为，所有发生过的经历通通翻篇，就是最好的放下，不值得分享，因为它们微不足道。最近看到一句话“40岁之前要从做有意义的事情中学会做人，40岁之后也学会做意义的人才能做正确的事。” 我想，也许生命对我有的一些启发，也许能启发你，我就完成了一件正确的事。假如世界进入倒计时……我还是那个“执意”要找到真爱并坚持以自己方式生活的人。这是我对自己和这个世界唯一的真实。大家如果私聊我可以加我微信哦一对一免费情感分析点阅读原文↓

# 一场家庭风波使我离开了喜爱的行业，现在很后悔怎么办


问：潘老师您好！我19岁来到上海工作，那时候的我有冲劲有目标，我努力自学英语和餐饮相关的知识，4年时间从一家知名西餐厅的跑菜员做到餐厅经理。但是发现自己并没有做服务员时候开心，我感觉压力很大，我擅长自我管理但不擅长管理别人，尤其是面对冲突时想要逃避，认为自己并不适合做管理层。2015年以身体不好需要休息为由辞职去了一家化妆品店做了销售，但工资变低了自己又开始焦虑不安。2016-2019年迫于生活压力又重回餐饮行业，做得也不错，工资和职位都有提升，但之前困扰我的问题依旧存在。2019年经历了家庭风波，离职给自己放了3个月的假来调整。稳定了情绪问题，开始解决工作问题，但是没有选择继续餐饮，依然选择了相对安逸的工作，只需管理好自己即可的那种。2020到现在一直从事文职助理方面工作，工资不高，工作轻松，一开始很满意，但近半年越来越有危机感，我的工作没有个人优势，觉得随时会被替代。今年我32岁了觉得这样下去非常危险，想要换份工作，但是没有方向，是不是该重回老本行呢？毕竟我有那么多年的餐饮服务行业的经验……潘幸知：这位同学的问题是关于职业生涯发展的，一方面担心自己的工作没有个人优势，会觉得被随时替代。另外一方面，觉得自己有一个干了很多年的餐饮服务行业的经验，但是并不愿意回去。那在你的问题当中，我看到几点：第一，我觉得你是一个有挑战愿望的人。为什么说有挑战的愿望？因为你有挑战的心，但是同时对挑战怀有惰性。你并不是一个安逸的人，如果只是安逸的工作，你会觉得人生很没有安全感。所以你不会适应永远安逸的工作，因为你会在这当中感觉到危机感。所以你的核心问题是面对进步型冲突时想要逃避，那么碰到困难的时候，如何解决冲突才是一个好的方式？如果你遇到的冲突在一年又一年的迭代，那就是一个好的解决方式。你在2012年之前遇到了困难，然后你跑了。今天到了2020年了，8年过去了，你还是再回到同样的一个原始问题上面绕圈。那如果这一次你的问题没有解决的话，也许再过10年你还会觉得说我今年我42岁了，我觉得这样下去十分危险，怎么办？第二，我觉得你对西餐厅的工作是最有感觉的。4年时间，你从一家知名的西餐厅的跑菜员做到餐厅经理，后来你又回到这个行业里面，这都说明你对这个行业是很有感觉的。我经常会看到有一些同学在某个行业里待久了，然后觉得对这个行业非常厌恶，说我再也不想做这个行业了，我要出来，但出来干了几年之后，又觉得，唉呀，我好像只会干那个行业，或者说干那个行业是最顺手的，也可能也是赚钱最多的，那到底要不要回到这个行业里面去？有感觉这件事情是非常重要的。比如说有些人在写作上面有天赋，他有这种感觉，他可能比别人花更少的时间去努力就能很擅长，但是他也有可能并不爱做这件事情。不过我认为感觉这个东西你一定要抓好他，它很重要。但为什么你会觉得不舒服？是因为在这个感觉里并不是一直都会顺利的，无论是做医生还是说做这个餐厅经理也好，包括你说做管理层，你发现困难一定是会有的，这个困难的出现，意味着我们在逃离自己的舒适区。有时候并不一定是我们自己厌恶这个行业了，而是我们发现这个困难像一个坎，我们没办法逾越过去，所以想逃避。觉得这个行业不好了，但我们自己的感觉还在这个行业里，所以我们要分清楚什么到底是感觉上的，什么是真正理性的选择。那从我看来的话，我觉得你是很适合餐饮这个行业的，只是你没有去跨过“不舒适区”。第三，尝试着跨越这个坎。你说了这10年的工作当中都是在做自我管理的事情。觉得自己管好就行了，但是不擅长去管理别人。从专家成长到管理者，这中间是有一个技能门槛的突破的。“管理”其实也是一个核心的技能，如果你今天在餐饮行业学习了管理，你到了文秘行业或者说人力行业的话，你也同样会做管理者。我们会看到很多的管理者，他在不同的行业之间都能够自如的切换。今天在餐饮行业，明天可能进入一个电商行业等等。那为什么？管理者这三个字本身就是一种专业能力。其实管理者并不一定在自己的这个专业能力上特别强，恰恰有的时候是专业能力上不够那么强，因为太强了他就会满足于在这个专业能力上的精进。管理者有一个最核心的能力，就是知道怎么样发挥其他人的优势去服务于一个目标。这就像阿基米德说的：给我一个杠杆和一个支点，我可以撬起地球。我知道这个支点在哪里，那我就把地球撬起来，管理者就是去寻找这个支点，所以如果你想做更大的事业的话，不妨让自己挑战一下。我可以推荐你读几本书，一本是管理学家德鲁克的《卓有成效的管理者》。还有台湾作家何飞鹏的《自慢》三部曲的最后一部，是讲怎么做管理的。那是我08年左右看的书，它们是我走向管理的启蒙之路。大家如果私聊我可以加我微信哦一对一免费情感分析点这里↓

# 家里有个很挑剔的人，怎么办


问：幸知老师，我老公特别挑剔，在家总会弄得鸡飞狗跳的，他说话喜欢阴阳怪气的，总说哪儿哪儿都不好，这里不好，那里不好。总是爱挑刺，话里话外不是挑剔我不会教育孩子，就是挑剔我不会勤俭持家。出一点事情，他会立刻把错误推到我一个人身上，让我有口难辩，相处起来真是太耗费心力了。不知道该怎么办？潘幸知：挑剔的人分三种类型。第一种类型，“通过挑剔，让你看见我”。这是一类披着完美主义外壳的爱人。我指出来的问题你没看见，但我看见了。我指出来的问题都是为了家里好，所以你要表扬我，这是这类挑剔者的内心所想。这一类人往往很委屈，付出不被看见，他是在用挑剔的方式来呼唤爱，但是如果我们把它解读为，他看我哪儿哪儿都不顺眼，那么我们就会特别的难受，但如果你解读为说，哦，他挑剔是为了更多的被肯定被认可被关注，那这事儿就好办多了。比如，有个姐妹说，我就感觉到很憋屈，搞了一下午的卫生，他还来一句，墙角还有几根头发，没有擦干净，我肯定生气啊，他自己就倒了下垃圾，还说这说那的。我们怎么说呢？老公，你真是火眼金睛！老婆看到了！那你拿抹布擦一下好不好？老婆收拾了好久有点累。我们直接说出对他的诉求。因为我们需要他的参与。遇到这类人，我们要在心里念叨“人事分开”，“对事不对人”。比如说他挑剔的只是房间里的卫生不干净，但不是挑剔我卫生做的不干净，更不是对我做了一下午卫生的否定。这样我们的感受会好很多。但如果一个男人挑剔仅仅只是因为自私，想给自己不干活找理由。那我们得找方法治治他了。想要治治这类男人，加幸知微信我们一起探讨。第二种类型叫做“我挑你毛病是为了你好”。你是自己人，所以我能挑剔你。因为挑剔你最安全。中国父母一直觉得挑剔都是为了孩子好，所以孩子在成长的过程中得不到认可和肯定，总是被挑剔，总是被迫成长，长大后就缺爱。面对这样的父母，我们要知道，他的挑剔不是针对你，而是他本身就是会很挑剔，挑剔这事儿他信手拈来，不要觉得是你不够好才被挑剔，要明白，挑剔是他的人格特质，与你无关，不必因为他的挑剔而怀疑自己，否定自己。当然，他的挑剔不会短期内消除，不要幻想一招用下去马上就可以改变，挑剔的人很容易引发别人的反抗，一反抗对方更挑剔，恶性循环不断，是非常容易惯性再有复发的。第三种类型叫做“对女性的挑剔”。这是一类“重男轻女型”挑剔。这样的挑剔无药可救，远离是正道。柳岩和她妈妈一起出席了一档综艺节目，在节目里，柳岩妈妈公开表示自己女儿年纪这么大，找个老头算了，周围人连忙圆场子，结果柳岩妈妈还在那挑剔女儿家里乱糟糟的、厨艺很差。这让柳岩在“全国人民面前出了丑”。但是，柳岩妈妈曾患有直肠癌，整个家庭无力承担高昂的医疗费，是柳岩甘愿背负骂名，疯狂接戏赚钱给她做手术。她为什么要在他人面前如此苛求自己的女儿呢？因为“我是她妈，她是我的，我当然可以挑剔她”。女性是赚钱的附属品，一切要为男性“弟弟”服务。帮助挑剔的人真正改变，这是一件非常困难的事情，但也不代表不可能。挑剔的背后是恐惧，害怕被否定，底层逻辑是在你否定我之前我先否定你。如果戴着怜悯的眼睛去看，很多看起来特别不好的特点，都是内心的苦太多了，没有方法化解导致的，当然，在帮助身边人的同时，我们自己也要去学习，因为先帮自己也等同于在帮家人。如果你家也有个挑剔的人，加幸知微信我们一起探讨。 大家如果私聊我可以加我微信哦

# 你的家庭结构图，揭示了亲密关系中的隐患

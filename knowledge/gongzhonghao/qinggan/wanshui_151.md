
# 以必死的决心去生活


拍婚纱照这件事，既是结婚的序曲，又是对爱情的阶段性总结，着实是一件值得投资和重视的事情，为广大新郎新娘们所喜闻乐见。谁都想把关于青春和爱情最美的一面留下来，固定在光影中，留待未来的回忆。不过也有比较另类的，直接想穿越时间隧道，一步跨到老年。这不，深圳的一位准新娘就让摄影师把自己画成70岁老太的样子，然后要求赶来的准新郎配合自己，也画个同样的妆容，拍一套基本等同于金婚的婚纱照。本来兴致勃勃的准新郎看到一个比自己丈母娘还老的“老女人”，差点没背过气去，直接拒绝，并要求准新娘洗脸重新画个正常妆。准新娘不肯，还问新郎：“到我70岁的时候你会不会爱我？”两个当街对吵，无果，准新郎气急败坏，落荒而逃，留下准新娘独自站在街头泪水横流，眼泪把脸上的“皱纹”都冲垮了。这真是一场现代都市的悲喜剧，叫人哭笑不得。有人说准新娘太“作”，有人怪准新郎没风度，说就结一回婚，就由着准新娘一次怎么了。我却觉得，如果这真的属于90后年轻一代的另类创意，是一种行为艺术还好办，问题是，她要的是考验。要以自己白发苍苍，一脸皱纹的暮年形象，来博得准新郎的一份爱情誓言：即使到70岁我也同样爱你。考验即意味着不信任，这是一切考验的本质。这种不信任是能够传递给对方的，会让对方产生本能的反感。矛盾就从这里产生，女的想的是，我不过就要的你一句话，你哪怕骗骗我也好啊；男的想的是，你既然不信任我，我保证了还有什么用？缺少安全感的女人，总是对生活和爱情都充满了过度的焦虑。这边还没有开启结婚的旅程呢，她就开始忧虑着：等我年老色衰的时候，这个男人会不会爱我。她要考验他，提前将生活的课题放在他面前。可对他来说，人家就是不可能娶个70岁老太太啊，也许共同生活的岁月可以将爱情凝结在生活的血肉之中，再也分不开，一个70岁的老头会爱着70岁的老太太，这没问题。但若是让20多岁的小伙子直接对着70岁的老太太说爱，那就是煎熬和难堪。女人总是热衷对男人进行各种幼稚的考验。前几天还看到个故事，第一次相亲，小伙子活活在约会地点等了一个多小时，姑娘才出现，解释说自己就是在考验他，看他有没有耐心。小伙子很生气，相亲之事泡汤了。这还是小事，还有人注册小号勾引自己男友，雇闺蜜测试老公忠诚度的，反正疑心生暗鬼，当信任出现裂痕，选择以考验的方式来获取最终答案的时候，答案就往往会不那么讨人喜欢。考验吧，考验吧，不是在考验中爆发，就是在考验中枯萎。在情感咨询中，很多女性问来问去，其实都是在问一个问题，那就是这个男人是不是可靠，能不能一辈子都一颗红心不褪色。是的，也许我们可以就男人的某些征兆来讨论一个概率问题，但就我们的一生来说，从来都没有万全的保证。我曾经问过一个极度忧虑“现在男人出轨这么多，我选的这个会不会变心”问题的姑娘：你想没想过，没准未来先变心的就是你呢，你凭什么给自己打包票？如果你觉得自己保证不了别人，其实你也同样保证不了自己。自己和别人都一样，都会在深不可测的岁月中行走出一段不可知的曲折的未来。他人即地狱，他人的薄弱也是我们的薄弱，我们同为人类，有着同样的脆弱和迷茫。爱情和生活，要想保持住一种品质，都是需要信仰的。信仰不可轻易动摇，让生活变得艰难的不止有变故，还有不能信任不敢信任的惶恐。千百年来，中国女人的焦虑一直都很难消除，她们怕男人变心，怕生活改变，怕孩子长大。是因为这种焦虑来自于内心，并不来自于外在。她们过多的向外界讨答案，不仅惯坏了男人，还迷失了自己。女人觉得自己要的只是男人的一句话，一个保证，用来弥补她们在爱情和生活中的不安全感。但这种非理性的行为，很难得到男人的配合，因为男人都是具有征服感的猛兽，当他们嗅到她们血液中的胆怯和畏缩，反而会令他们变得更加骄傲和血腥。不要人为的考验，不要刻意的考验，如果有足够智慧，生活自会提供让我们去了解的机会。日子总要一天天过，没有人能够直接跨越时间就看到结果，有这个本领的人会疯掉。卫斯理的小说就曾经写过这样一个人，他知道自己会死于何年何月，以何种方式，他看自己的每一天就像一张旧报纸，了无生趣。生活叫人迷惑之处就在于我们明知正逐渐接近死亡但仍兴致勃勃，创造各种快乐与惊喜。爱情也是一样，即使有一天不可避免的会离散，但谁也不能阻挡我们分享今天的幸福。如果不死，就一定能够活到70岁。如果不分手，就一定能够白头偕老。全都是废话，毫无意义。然而活着本身就是一种黑色幽默，要坚强，要自嘲，才能在湍急的时间洪流中保持住自信和平静。我们生命中的挫折有必然的，也有偶然的。必然的挫折寄生在自己的缺陷身上，偶然的挫折由命运随机分配，那么，修炼好自己解决掉必然的，耐着性子等待偶然的，以必死的决心去认真生活，就不会再有那么多不安。查看历史消息可收看往期文章。阅读原文可以进入我的新书《你配得起更好》京东购买界面。温馨小提示：喜欢这篇文章的人可以到第二篇情感咨询文下点赞赏，多谢支持。

# 以成熟的心态为亲情续费


讲述： 晚睡姐姐，你好，我最近一直在困扰于亲情，我从小没有爸妈哥哥姐姐生活在一起，五岁才回到她们身边，哥哥姐姐年龄长我有十多岁，自小就没有和我玩，没有和我亲密过，所以现在长大了感情也不懂得去联系，我妈妈就说我说读那么多年的书的白读了，你大学人家都有给你寄过生活费，你平常连个电话都不懂得给哥哥姐姐打问个好，说我没用，真是一点事都不懂，是我知道我在这方面真是弱点，可是我也会想，都说是我哥哥姐姐了，比我大那么多岁，为什么她们就不给我打过电话嘘寒问暖，打电话不是问我有没有乱花钱就是问我有没有找兼职做，我知道谁都不容易，可是还是很纠结为什么身边朋友的家里都和乐融融的，亲情之间都处理的那么好，我又该怎么处理这些呢？ 任何人际关系的本质都是互动。你和你周边的人相互给予，能量和情感进行交换和流动。当这种交换和流动能保持基本平衡的时候，你的人际关系大抵上就是良好的。一旦这个平衡打破，人际关系就会变得混乱，失控，无序，同时，也会极大地影响一个人的生活质量。我们中国人习惯强调血缘亲情的天然性，觉得只要有亲缘关系，就不管是圆是扁，都硬往一块捏。进入现代社会，这种看法显然已经不符合人性，亲情也是人际关系的一种，同样需要经营，也必须要遵守人际关系的法则。而人际交往的第一黄金法则就是，你想要得到什么，就必须先给出去什么。对这一点，人们常常会有误区，很多人总是先看到自己的缺乏，然后去别人那里求，求不到就觉得失望。这样其实首先证明自己内在情感的匮乏，其次是将自己处于被动的境地，不能成为人际关系中的主导。这里所说的“给”，并非指的是物质和金钱上的付出，而是关怀、理解、尊重、爱、善意等正能量，尤其是亲人之间，物质只是承载情感的手段，而最终要传递的，始终还是温情。所以，虽然你的哥哥和姐姐和你一样，没有在一起生活过，没有形成亲密无间的氛围，但他们却用给寄生活费这样的方式在表达对你的关心和爱。这是他们的“给”。而你呢，如果你想再进一步，继续营造你们之间的亲情，也就应该给予回应，给出自己的反馈。打个电话，问声好，说句谢谢，等等。只有这样，你们之间的情感和能量才能流动起来，形成良性的循环反应。在你所看不见的别人家亲密关系的下面，都有这种互动。《诗经》中说：“投我以木桃，报之以琼瑶。匪报也，永以为好也。”——你用木桃投赠我， 我拿琼瑶作回报。不是为了答谢你，而是我珍惜这份情谊，希望我们永远交好下去。我家姐弟三人，弟弟要搬家，打电话：“我把儿子送到你家里住几天。”我说好呀，求之不得。我举家旅游，姐姐搬进我家替我照顾猫咪，正巧猫咪生病，她每天奔波两个小时带它去打针看病，无怨无悔。人字的结构就是相互支撑，我们都希望被人需要也需要别人，兄弟姐妹排在全世界所有的人面前，是一个最适当的选项。你们的亲情没有最好的开始，但可以在大家都成年之后用成熟的心态为它“续费”。不必按照别人家的标准有过高的要求和心理预期，珍惜每一点细微的友好，主动去修复和弥补过去的缺失，这才是一个渴望亲情的人应该去做到的。查看历史消息可收看往期文章。阅读原文可以进入我的新书《你配得起更好》京东购买界面。

# 小委屈不忍，大委屈必来

# 分手后，最是思念无处投寄

# 一切都将由自己开始，也将由自己收场


生活中，总能遇到一些脾气大的人，忍不了火，三言两语不和，就吵将起来。事后沟通，“你为啥这么冲动啊？”也总是说：“我控制不住自己啊。”我有个前同事就是这种人。她当时的工作是坐窗口接待客户，实事求是地说，客户中真是三教九流都有，素质差的也不少，稍不留神，有的人就开始摔摔打打，又吵又闹。但不管客户如何，单位的岗位规范是要求坚决不能和客户吵架的。所以别的同事该闭嘴就闭嘴，等客户发泄完情绪，再耐心解释一下，基本上都能做到让客户满意。只有她偏不，像刘胡兰一样英勇不屈，往往客户还没急她先急了，客户要是再敢啰嗦几句，她甚至会忍不住开始动手、丢东西，什么水杯、印台、算盘，时不时的就会从她的窗口飞出来。有一次，她居然拎着凳子从柜台后面跳出来，追得客户满大厅跑。幸亏啊，90年代客户的维权意识还那么强，群众还是比较淳朴的，所以对骂了几句就很怂包的溜掉了，没有酿成什么血光之灾。我们都帮她瞒住，不希望领导处罚她，可是总得劝劝她。“亲爱的，你别这样，要下岗的。”“下岗就下岗，我无所谓。”梗着脖子，一副我为真理而死死得其所的样子。“值得吗？”“值得。”哪里值得了，完全不值得。只是情绪上来的时候，没有理性思考的能力。就这脾气，让她吃了很多年的亏。职位总也升不上去，连身体都受影响了——成天像个炸弹似的爆发可是需要很大能量的，那么投入的愤怒、对峙，每一次都是一场身心的浩劫，久而久之，必定有损健康。这是工作中的暴脾气，还有的人是和自己的家人、朋友、伴侣发火，明明是最亲的人，却偏偏按照对待阶级敌人的方式，毫无包容之心，苛刻、打击、诋毁。像炮仗似的，一点就着。每每搞砸了之后又后悔，后悔完了一切照旧。“我控制不住走自己”——是这类人的统一的招牌。是真的控制不住吗？也未必。人们发火，大多目的都是想要维护自己，不想被别人欺负和贬低，但情绪其实解决不了问题，而且发出的那些火，其实最后都会还到自己身上。比如文章，明星中有名的坏脾气，在各种真真假假的明星爆料中，别人的形象或者个性可能都各有出入，是好是坏，不固定，只有他，始终如一的暴躁、粗鲁、耍大牌、骂脏话、爱动手。说他家暴马伊琍，有人信；说他不吃油泼面不上台，有人信；说他怒骂粉丝，有人信……常言说不作不死，他是真的把自己作死了之后，干什么都是死路一条。所以当他出轨被偷拍，全国的媒体和娱记几乎都没有愿意帮他说话的，全都恨不得上去再踩一脚。现在的文章低调谨慎，得到了足够教训的他，大概对于今后的人生并不是一件坏事。儿子每天午饭都在附近的一所大学的食堂解决，我给他办了一张卡。前几天他回家对我讲了一件事情。我转述过来，是这样的：我今天和同学一起去吃饭，同学脱下校服，放在桌子上，然后我们两个人去选菜。选好回来，却发现校服被丢到一边，座位上坐了一个四五十岁的女人正在吃饭。我说阿姨，这是我们的位置，你看我们的校服还在这里呢。这位阿姨很不友善地说，哪又怎么样，我影响你们了？我说可我们是两个人。阿姨很不高兴，说谁让你们占座的，然后嘴里不干不净地开始骂我们没素质。我的火腾地一下子就起来了！我真想痛斥她，和她吵一架。可是心里又一想，第一，这里不是我们学校，不是我的底盘，我容易吃亏，第二，这老太太看起来不仅素质差，好像精神还有点问题，万一吵起来，我可控制不住事态发展。所以，我什么都没说，拿起衣服就走了，和同学换了一个位置吃了这顿饭。孩子学得有声有色，尤其是那段心理活动，跟演戏似的，我一边听一遍笑个不停。这孩子平时很随和，可犯起倔来，十头牛都拉不回来，每次发作都能把我气得半死。可现在居然也知道制怒了，真的是长大了。我和儿子说：“这就是我常教你的，话少说一句，就可能会逆转整个事态的发展。”他说是啊，我没和她吵，那点不愉快和同学高高兴兴吃完这段饭就忘掉了，但我要是吵了这一顿，估计现在还会不开心。以前我妈总是对我说：“不要愁孩子不懂事，孩子懂事也就是一件事两件事上。”那时不相信，现在我信了，我眼里那个胖乎乎笑眯眯的男孩，经过这件事，正在成长为一个有思想的成熟青年。在他身上，我突然意识到，成熟，其实就是能管得住自己。过去他想吵就吵，想发火就发火，是因为没有真正养成自己为自己负责的精神，反正哭了闹了，有父母哄，外面闯祸了，也有父母出面摆平。等意识到，一切都将由自己开始，也将由自己收场，所以才会思考“这里不是我的底盘，对我不利”这种虽然幼稚，但能管控得住自己的理由。收敛自己的脾气，是做人的修养，也是明白了自己必须为自己负责任之后的一种理性。如果缺乏这种理性和责任心，就会变成只知道开启一段暴怒的情绪，却不知道如何收场，搞得一地鸡毛之后，默默承担后果。郭芙蓉脾气不好，吕秀才教她每次发火之前都要默念：“世界如此美妙，我却如此暴躁，这样不好，不好。”其实世界并非那么美妙，总有各种不如意，所以更需要我们用理性和修养来承接这些不如意，“世界如此暴躁，我却如此美妙，这样真好，真好”。 # 并非一切婚姻矛盾的根源都在于穷


讲述：您好，非常喜欢您的文章，一直在关注，现在我和我结婚七八年的老婆关系一直不稳定，可能麻烦指导下啊。我是来自农村清贫的多子女家庭，她是城市独生女，我是内向，她外向，结婚时我没买房给了她家十万就住在了她家和她父母生活，以后就是我俩矛盾不断，和她妈关系也不如以前好，结婚三四年后我们共同凑钱做了个生意，现在赚钱了她说都是她挣的，我上班拿不到高工资，不仅她大权在握，还贬低我和我的家人，不和我分享胜利成果，我想用钱没有门，更不能支持我做生意，我现在不喜欢她但是我父母反对离，我现在过得好压抑。 刘震云的小说《一地鸡毛》中，主人公小林就是一个农村出身，大学毕业后留在北京某部门工作的小职员。他老婆是城里人，选定他是因为他在城市所扮演的角色，并未意识到城乡差别能够对婚姻带来哪些影响。结婚之后才发现，问题太多了，家里不时有小林老家的乡亲来投奔。来了就要招待，招待就得加菜，加菜伙食费就要超标。如果仅仅是招待吃一顿饭，多点花费还好办，问题是“吃过饭，他们还要交代许多事让小林办。搞物资、搞化肥、买汽车，打官司，走时还让小林给买火车票。”这让小林在老婆面前抬不起头来，“人家家老不来人，自己家老来人，来了就要吃饭，农村人又不讲究，到处弹烟灰吐痰。”开始的时候小林老婆还忍着，后来次数一多，表面功夫都不愿意做了，“来了客人就脸色不好，也不去买菜，也不去下厨房。”长期以往，两个都曾经不食人间烟火，眉清目秀的单纯青年，变成了一对伧俗算计的小市民夫妇。当然，距离刘震云写《一地鸡毛》的年代已经过去了三十年，城乡之间的差距已经在缩小，比如江浙一带的有些农村，比城市还要富裕，豪车满地跑，随随便便嫁个女儿，嫁妆就百十来万，这样的农村人，哪里还有人介意你是什么身份。估计你也是这么想的吧，心里憋着一口气，“你们还不是嫌弃我是农村人，家里没钱吗？”真的是没钱的问题吗？不，结婚能拿出十万块钱的人算不得真正的穷人。你不穷，你只是有自己的小算盘，给娘家十万块然后换来和岳父母一起住，一举两得，彩礼听起来也体面，自己也有房子住了。但这种做法的危害是埋下了后患，对方始终都会觉得你是占便宜的一方，结婚了，姑娘没嫁出去，反倒领回一个回家住，那些有虚荣心的父母都会觉得抬不起头来。即使当初这种选择可以达成某种平衡，时间长了，也会觉得吃亏。莫不如当初用这钱做首付，自己赚钱买房子，吃苦点，不贪便宜，也不会落人口实。一直住在家里的女孩，很难成长，始终还是处于“我是父母孩子”的身份认同之中。父母对她依旧施加着各种影响，对你老婆来说，你这个老公不是用来雪中送炭，是用来锦上添花的，有你五八，没你四十。你稍有不足，她就会想，我本来一直都过得挺好，有爹妈照顾，我为何要找一个不合适的老公给自己堵心？你说了，你又是那种内向，不擅长表达的男人，当你老婆想：我可以不介意你们家穷，我图的是感情，你对我好点也行，你想的可能却是：本来我就感觉自己抬不起头来，要是再说软话，岂不是更没尊严了吗？你们如盲人摸象一把，各奔着自己想象中的方向而去。这种内向，是农村孩子身上缺乏情感教育的外在表现，父母没有教过如何为人处世，家庭缺少温情，让孩子不会说也不擅长说窝心的话，越是觉得自己理亏的时候，反而越会用沉默用强硬来保护自己。比如我农村有个亲戚，父母都是好人，含辛茹苦的每日奔波，给儿子创造了比较优裕经济环境，可是他们之间，每天连几句话都聊不上，只知道孩子缺什么给什么。孩子的智商很高，但情商极其低下，不知道如何和人交往，连和旁人打招呼都是一件特别困难的事情，放学忘带钥匙，宁可在雨站着，也不肯去邻居家敲门说我可以进去躲一会吗。他后来读到了大学，有体面的身份，可我时常担心，他未来的情感生活怎么办？他知道怎么和伴侣沟通和交流吗？他已经习惯了像河蚌一样紧紧关闭自己的心门，又有多少女人有那样的耐心和智慧去撬动这扇门呢？即使如谢耳朵那样的天才，艾米最后都忍受不了他的愚钝和不解风情，他分手了。农村有落后传统，城里也有小市民，二者没有谁比谁高贵，都各自有各自的局限，最终，很多城乡模式的破裂，就在于无法建立一个通畅的沟通交流渠道，双方的成长环境不同，对于情感有着不同的理解和需求。并非一切婚姻矛盾的根源都在于穷，只不过人们习惯用穷来解释，这样最方便易懂，还能占据道德优势，“你还不就是嫌我穷？”言外之意，你就是一个贪心的女人。《一地鸡毛》中，小林由衷感慨，“老家如同一个大尾巴，时不时要掀开让人看看羞处，让人不忘记你仍是一个农村人。”你拖着的这个尾巴，就是出身带给你的性格缺陷，就是你的不擅长沟通，而令你的老婆有一种觉得钱和人自己都没图到的失落。试着换一种思路去经营夫妻关系，外向的女孩其实都没有什么坏心眼，她当初能选择你，也未必非常介意你的出身，更可能是看重你的老实可靠。但仅有老实可靠，又会令人感到无趣，要是能学会多表达，多赞美，多鼓励，哄得她高高兴兴的，很多事情都会简单得多。阅读原文可以进入我的新书《你配得起更好》京东购买界面。点击右下角的往期文章可以进入文章分类链接。
